## Авто

Используйте `h-auto`, чтобы браузер определял высоту элемента.

```html amber
<template preview>
  <div class="h-auto w-40 mx-auto rounded-md bg-amber-500 text-white text-2xl font-extrabold flex items-center justify-center p-6">h-auto</div>
</template>

<div class="**h-auto** ...">h-auto</div>
```

## Высота экрана

Используйте `h-screen`, чтобы элемент занимал всю высоту области просмотра.

```html rose
<template preview>
  <div class="h-screen mx-auto rounded-md bg-rose-500 text-white text-2xl font-extrabold flex items-center justify-center p-6">h-screen</div>
</template>

<div class="**h-screen** p-6 ...">h-screen</div>
```

## Фиксированная высота

Используйте `h-{number}` или `h-px`, чтобы установить элемент на фиксированную высоту.

```html indigo
<template preview>
    <div class="flex justify-around items-end space-between-3">
        <div>
            <div class="h-8 w-8 rounded-md bg-indigo-500 text-white text-2xl font-extrabold"></div>
            <p class="text-center mt-2 text-sm text-indigo-600">h-8</p>
        </div>
        <div>
            <div class="h-12 w-8 rounded-md bg-indigo-500 text-white text-2xl font-extrabold"></div>
            <p class="text-center mt-2 text-sm text-indigo-600">h-12</p>
        </div>
        <div>
            <div class="h-16 w-8 rounded-md bg-indigo-500 text-white text-2xl font-extrabold"></div>
            <p class="text-center mt-2 text-sm text-indigo-600">h-16</p>
        </div>
        <div>
            <div class="h-24 w-8 rounded-md bg-indigo-500 text-white text-2xl font-extrabold"></div>
            <p class="text-center mt-2 text-sm text-indigo-600">h-24</p>
        </div>
    </div>
</template>

<div>
    <div class="**h-8** ..."></div>
    <div class="**h-12** ..."></div>
    <div class="**h-16** ..."></div>
    <div class="**h-24** ..."></div>
</div>
```

## Полная высота

Используйте `h-full`, чтобы установить высоту элемента равной 100% от его родительского элемента, если родительский элемент имеет определенную высоту.

```html
<template preview class="flex justify-around items-end text-sm p-4">
  <div class="h-48">
    <div class="h-full w-40 rounded-md bg-indigo-500 text-white text-2xl font-extrabold flex items-center justify-center p-6">h-full</div>
  </div>
</template>

<div class="h-48">
  <div class="**h-full** ...">h-full</div>
</div>
```

## Адаптивность

Чтобы контролировать высоту элемента в определенной контрольной точке, добавьте префикс `{screen}:` к любой существующей утилите ширины. Например, добавление класса `md:h-full` к элементу приведет к применению утилиты `h-full` при средних размерах экрана и выше.

```html
<div class="h-8 **md:h-full**"></div>
```

Для получения дополнительной информации о функциях адаптивного дизайна Tailwind ознакомьтесь с документацией [Адаптивного дизайна](/docs/responsive-design).

## Кастомизация

### Шкала высоты

По умолчанию шкала высоты Tailwind представляет собой комбинацию [шкалы интервалов по умолчанию](/docs/customizing-spacing#default-spacing-scale), а также некоторых дополнительных значений, специфичных для высот.

Вы можете настроить шкалу интервалов для заполнения, полей, ширины и высоты одновременно в разделе `theme.spacing` Вашего файла `tailwind.config.js`:

```diff-js
  // tailwind.config.js
  module.exports = {
    theme: {
      spacing: {
+       sm: '8px',
+       md: '16px',
+       lg: '24px',
+       xl: '48px',
      }
    }
  }
```

Чтобы настроить высоту отдельно, используйте раздел `theme.height` Вашего файла `tailwind.config.js`.

```diff-js
  // tailwind.config.js
  module.exports = {
    theme: {
      height: {
+       sm: '8px',
+       md: '16px',
+       lg: '24px',
+       xl: '48px',
      }
    }
  }
```
