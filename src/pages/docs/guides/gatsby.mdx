```preval setup
dependencies:
  - gatsby-plugin-postcss
```

```preval configuration
purge:
  - ./src/**/*.{js,jsx,ts,tsx}
```

### Включить gatsby-plugin-postcss

В вашем файле `gatsby-config.js`, включите [плагин postcss](https://www.gatsbyjs.com/plugins/gatsby-plugin-postcss/).

```diff-js
  // gatsby-config.js
  module.exports = {
    /* Your site config here */
-   plugins: [],
+   plugins: ['gatsby-plugin-postcss'],
  }
```

```preval include
create: true
file: ./src/styles/global.css
```

Наконец, создайте файл `./gatsby-browser.js` в корне вашего проекта, если он еще не существует, и импортируйте свой файл CSS:

```diff-js
  // ./gatsby-browser.js
+ import './src/styles/global.css';
```

Прочтите [документацию Gatsby по использованию глобальных стилей](https://www.gatsbyjs.com/docs/recipes/styling-css/#using-global-css-files-without-a-layout-component), чтобы узнать больше о работе с глобальными файлами CSS в Gatsby.
