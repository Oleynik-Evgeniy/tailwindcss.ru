## Применение

Установите минимальную высоту элемента с помощью утилит `min-h-0`, `min-h-full` или `min-h-screen`.

```html purple
<template preview>
  <div class="h-48 p-6 bg-purple-300 rounded-md">
    <div class="h-24 min-h-full p-6 bg-purple-500 flex items-center justify-center rounded text-white font-semibold">
      <span>.min-h-full</span>
    </div>
  </div>
</template>

<div class="h-48 ...">
  <div class="h-24 **min-h-full** ...">
    .min-h-full
  </div>
</div>
```

---

## Адаптивность

Чтобы управлять минимальной высотой элемента в определенной точке останова, добавьте префикс `{screen}:` к любой существующей утилите минимальной высоты.

```html
<div class="h-48 ...">
  <div class="h-24 min-h-0 **md:min-h-full** ...">
    <!-- ... -->
  </div>
</div>
```

Для получения дополнительной информации о функциях адаптивного дизайна Tailwind ознакомьтесь с документацией [Адаптивного дизайна](/docs/responsive-design).

---

## Кастомизация

### Шкала минимальной высоты

Настройте шкалу минимальной высоты по умолчанию для Tailwind в разделе `theme.minHeight` Вашего файла `tailwind.config.js`.

```diff-js
  // tailwind.config.js
  module.exports = {
    theme: {
      minHeight: {
+       '0': '0',
+       '1/4': '25%',
+       '1/2': '50%',
+       '3/4': '75%',
+       'full': '100%',
      }
    }
  }
```

Дополнительные сведения о настройке темы по умолчанию смотрите в [документации по настройке темы](/docs/theme#customizing-the-default-theme).
