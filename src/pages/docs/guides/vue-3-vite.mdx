---
title: "Установите Tailwind CSS с Vue 3 и Vite"
description: "Настройка Tailwind CSS в проекте Vue 3 и Vite."
tool: Vue 3 и Vite
---

```preval installation
tool: Vite
script: npm init @vitejs/app
npmInstall: true
```

---

```preval setup
version: latest
```

```preval configuration
purge:
  - ./index.html
  - ./src/**/*.{vue,js,ts,jsx,tsx}
```

```preval include
tool: Vite
file: ./src/index.css
create: true
```

Наконец, убедитесь, что ваш файл CSS импортируется в файл `./src/main.js`:

```js
// src/main.js
import { createApp } from 'vue'
import App from './App.vue'
import './index.css'

createApp(App).mount('#app')
```

---

```preval finish
scripts:
  - npm run dev
```
