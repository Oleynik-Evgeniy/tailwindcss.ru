## Применение

Переведите элемент, включив сначала преобразования с помощью утилиты `transform`, а затем указав направление перевода и расстояние с помощью утилит `translate-x-{amount}` и `translate-y-{amount}`.

```html purple
<template preview>
  <div class="flex flex-col justify-around space-y-12 md:flex-row md:space-y-0 md:space-x-12">
    <div class="w-16 h-16 bg-purple-400 bg-stripes bg-stripes-white">
      <img class="h-16 w-16 transform translate-y-6" src="https://images.unsplash.com/photo-1472099645785-5658abf4ff4e?ixlib=rb-1.2.1&ixid=eyJhcHBfaWQiOjEyMDd9&auto=format&fit=facearea&facepad=2&w=256&h=256&q=80">
    </div>
    <div class="w-16 h-16 bg-purple-400 bg-stripes bg-stripes-white">
      <img class="h-16 w-16 transform -translate-y-6" src="https://images.unsplash.com/photo-1472099645785-5658abf4ff4e?ixlib=rb-1.2.1&ixid=eyJhcHBfaWQiOjEyMDd9&auto=format&fit=facearea&facepad=2&w=256&h=256&q=80">
    </div>
    <div class="w-16 h-16 bg-purple-400 bg-stripes bg-stripes-white">
      <img class="h-16 w-16 transform translate-y-0" src="https://images.unsplash.com/photo-1472099645785-5658abf4ff4e?ixlib=rb-1.2.1&ixid=eyJhcHBfaWQiOjEyMDd9&auto=format&fit=facearea&facepad=2&w=256&h=256&q=80">
    </div>
  </div>
</template>

<img class="transform **translate-y-6** ...">
<img class="transform **-translate-y-6** ...">
<img class="transform **translate-y-0** ...">
```

Обратите внимание: поскольку Tailwind реализует преобразования с использованием [настраиваемых свойств CSS](https://developer.mozilla.org/en-US/docs/Web/CSS/Using_CSS_custom_properties), **утилиты преобразования не поддерживаются в старых браузерах, таких как IE11**. Если вам нужны преобразования для вашего проекта и вам нужна поддержка старых браузеров, [добавьте свои собственные утилиты](/docs/adding-new-utilities) или другой настраиваемый CSS.

## Адаптивность

Чтобы контролировать перевод элемента в определенной контрольной точке, добавьте префикс `{screen}:` к любой существующей утилите перевода. Например, используйте `md:translate-x-8`, чтобы применить утилиту `translate-x-8` только при средних размерах экрана и выше.

```html
<img class="transform translate-x-2 **md:translate-x-8** ...">
```

Для получения дополнительной информации о функциях адаптивного дизайна Tailwind ознакомьтесь с документацией [Адаптивного дизайна](/docs/responsive-design).

## Кастомизация

### Шкала перевода

По умолчанию Tailwind предоставляет утилиты перевода с фиксированным значением, которые соответствуют нашему [шкале интервалов](/docs/customizing-spacing), а также 50% и 100% вариации для перевода относительно размера элемента.

Вы можете настроить глобальную шкалу интервалов в разделах `theme.spacing` или `theme.extend.spacing` вашего файла `tailwind.config.js`:

```diff-js
  // tailwind.config.js
  module.exports = {
    theme: {
      extend: {
        spacing: {
+         '72': '18rem',
+         '84': '21rem',
+         '96': '24rem',
        }
      }
    }
  }
```

Чтобы настроить масштаб перевода отдельно, используйте раздел `theme.translate` вашего файла `tailwind.config.js`.

```diff-js
  // tailwind.config.js
  module.exports = {
    theme: {
      extend: {
        translate: {
+         '1/7': '14.2857143%',
+         '2/7': '28.5714286%',
+         '3/7': '42.8571429%',
+         '4/7': '57.1428571%',
+         '5/7': '71.4285714%',
+         '6/7': '85.7142857%',
        }
      }
    }
  }
```

Дополнительные сведения о настройке темы по умолчанию смотрите в [документации по настройке темы](/docs/theme#customizing-the-default-theme).
