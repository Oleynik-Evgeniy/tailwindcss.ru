## Auto

Используйте `place-self-auto` для выравнивания элемента на основе значения свойства контейнера `place-items`:

```html emerald
<template preview>
  <div class="grid grid-cols-3 gap-2 place-items-stretch h-48">
    <div class="rounded-md bg-emerald-300 text-white text-2xl font-extrabold flex items-center justify-center py-3 px-6">1</div>
    <div class="place-self-auto rounded-md bg-emerald-500 text-white text-2xl font-extrabold flex items-center justify-center py-3 px-6">2</div>
    <div class="rounded-md bg-emerald-300 text-white text-2xl font-extrabold flex items-center justify-center py-3 px-6">3</div>
    <div class="rounded-md bg-emerald-300 text-white text-2xl font-extrabold flex items-center justify-center py-3 px-6">4</div>
    <div class="rounded-md bg-emerald-300 text-white text-2xl font-extrabold flex items-center justify-center py-3 px-6">5</div>
    <div class="rounded-md bg-emerald-300 text-white text-2xl font-extrabold flex items-center justify-center py-3 px-6">6</div>
  </div>
</template>

<div class="grid grid-cols-3 gap-2 ...">
  <div>1</div>
  <div class="**place-self-auto** ...">2</div>
  <div>3</div>
  <div>4</div>
  <div>5</div>
  <div>6</div>
</div>
```

## Адаптивность

Чтобы поместить элемент в определенную контрольную точку, добавьте префикс `{screen}:` к любой существующей утилите `place-self`. Например, используйте `md:place-self-end`, чтобы применить утилиту `place-self-end` только при средних размерах экрана и выше.

```html
<div class="place-self-start md:place-self-end">
  <!-- ... -->
</div>
```

Для получения дополнительной информации о функциях адаптивного дизайна Tailwind ознакомьтесь с документацией [Адаптивного дизайна](/docs/responsive-design).
