## <Heading hidden>Утилита прежде всего</Heading>

Они поставляются со всевозможными предварительно разработанными компонентами, такими как кнопки, карточки и предупреждения, которые могут помочь Вам сначала двигаться быстро, но причинят больше боли, чем облегчения, когда придет время выделить Ваш сайт индивидуальным дизайном.

Tailwind различен.

Вместо самоуверенных предварительно разработанных компонентов Tailwind предоставляет низкоуровневые служебные классы, которые позволяют создавать полностью индивидуальные проекты, не покидая своего HTML.

```html
<template preview class="p-8">
  <div class="md:flex">
    <div class="md:flex-shrink-0">
      <img class="rounded-lg md:w-56" src="https://images.unsplash.com/photo-1556740738-b6a63e27c4df?ixlib=rb-1.2.1&ixid=eyJhcHBfaWQiOjEyMDd9&auto=format&fit=crop&w=448&q=80" alt="Woman paying for a purchase">
    </div>
    <div class="mt-4 md:mt-0 md:ml-6">
      <div class="uppercase tracking-wide text-sm text-indigo-600 font-bold">Маркетинг</div>
      <a href="#" class="block mt-1 text-lg leading-tight font-semibold text-gray-900 hover:underline">Поиск клиентов для вашего нового бизнеса</a>
      <p class="mt-2 text-gray-600">Начало нового бизнеса - это тяжелая работа. Вот пять идей, которые Вы можете использовать, чтобы найти своих первых клиентов.</p>
    </div>
  </div>
</template>
```

Если Вам надоело бороться с фреймворком, игнорировать нежелательные стили и бороться за специфичность, Tailwind создан для Вас.

## Отзывчивость до мозга костей

Каждая утилита Tailwind также поставляется с адаптивными вариантами, что значительно упрощает создание адаптивных интерфейсов без использования настраиваемого CSS.

Tailwind использует интуитивно понятный префикс `{screen}:`, который позволяет легко заметить отзывчивые классы в Вашей разметке, сохраняя при этом исходное имя класса узнаваемым и неповрежденным.

<ResponsiveCodeSample
  classNames={['justify-start', 'justify-center', 'justify-end', 'justify-between', 'justify-around']}
  preview={(className) => (
    <div className={`flex bg-gray-200 ${className}`}>
      <div className="text-gray-700 text-center bg-gray-400 px-4 py-2 m-2">1</div>
      <div className="text-gray-700 text-center bg-gray-400 px-4 py-2 m-2">2</div>
      <div className="text-gray-700 text-center bg-gray-400 px-4 py-2 m-2">3</div>
    </div>
  )}
/>

## Дружественность с компонентами

Хотя Вы можете *много* делать только с служебными классами, по мере роста проекта может быть полезно кодифицировать общие шаблоны в абстракции более высокого уровня.

Tailwind предоставляет инструменты для [извлечения классов компонентов](/docs/extracting-components) из повторяющихся служебных шаблонов, что упрощает обновление нескольких экземпляров компонента из одного места:

```html
<template preview class="p-4 text-center">
  <button class="bg-blue-500 hover:bg-blue-600 text-white font-bold py-2 px-4 rounded">
    Кнопка
  </button>
</template>

<!-- Использование утилит: -->
<button class="bg-blue-500 hover:bg-blue-600 text-white font-bold py-2 px-4 rounded">
  Кнопка
</button>

<!-- Извлечение классов компонентов: -->
<button class="btn btn-blue">
  Кнопка
</button>

<style>
  .btn {
    @apply font-bold py-2 px-4 rounded;
  }
  .btn-blue {
    @apply bg-blue-500 text-white;
  }
  .btn-blue:hover {
    @apply bg-blue-600;
  }
</style>
```

[Узнайте больше об извлечении компонентов с помощью Tailwind &rarr;](/docs/extracting-components)

## Создан для настройки

Если есть смысл настраивать, Tailwind позволяет это. Это включает в себя цвета, размеры границ, толщину шрифтов, утилиты интервалов, точки останова, тени и многое другое.

Tailwind написан на [PostCSS](http://postcss.org/) и настроен на JavaScript, что означает, что у Вас под рукой вся мощь настоящего языка программирования.

Tailwind - это больше, чем CSS-фреймворк, *это движок для создания дизайн-систем.*

```js
// tailwind.config.js
module.exports = {
  theme: {
    screens: {
      tablet: '768px',
      desktop: '1024px',
    },
    colors: {
      primary: {
        100: '#ebf8ff',
        300: '#90cdf4',
        500: '#4299e1',
        700: '#2b6cb0',
        900: '#2a4365',
      },
      secondary: {
        100: '#fffff0',
        300: '#faf089',
        500: '#ecc94b',
        700: '#b7791f',
        900: '#744210',
      },
    },
    extend: {
      boxShadow: {
        huge: '0 30px 60px -15px rgba(0, 0, 0, .25)'
      }
    }
  }
}
```
