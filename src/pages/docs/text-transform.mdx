## Нормальный регистр

Используйте утилиту `normal-case`, чтобы сохранить исходный корпус. Обычно это используется для сброса заглавных букв в разных контрольных точках.

```html emerald
<template preview>
  <p class="normal-case text-xl text-emerald-600">Быстрая, коричневая лиса, перепрыгнула через ленивого пса.</p>
</template>

<p class="**normal-case** ...">Быстрая коричневая лиса...</p>
```

## Верхний регистр

Используйте утилиту `uppercase` для текста в верхнем регистре.

```html purple
<template preview>
  <p class="uppercase text-xl text-purple-600">Быстрая, коричневая лиса, перепрыгнула через ленивого пса.</p>
</template>

<p class="**uppercase** ...">Быстрая коричневая лиса...</p>
```

## Строчные буквы

Используйте утилиту `lowercase` для текста в нижнем регистре.

```html rose
<template preview>
  <p class="lowercase text-xl text-rose-600">Быстрая, коричневая лиса, перепрыгнула через ленивого пса.</p>
</template>

<p class="**lowercase** ...">Быстрая коричневая лиса...</p>
```

## Заглавные буквы

Используйте утилиту `capitalize` для написания текста с заглавной буквы.

```html fuchsia
<template preview>
  <p class="capitalize text-xl text-fuchsia-600">Быстрая, коричневая лиса, перепрыгнула через ленивого пса.</p>
</template>

<p class="**capitalize** ...">Быстрая коричневая лиса...</p>
```

## Адаптивность

Чтобы управлять преобразованием текста элемента в определенной контрольной точке, добавьте префикс `{screen}:` к любой существующей утилите преобразования текста. Например, используйте `md:uppercase`, чтобы применить утилиту `uppercase` только при средних размерах экрана и выше.

```html
<p class="capitalize **md:uppercase** ...">
  Быстрая, коричневая лиса, перепрыгнула через ленивого пса.
</p>
```

Для получения дополнительной информации о функциях адаптивного дизайна Tailwind ознакомьтесь с документацией [Адаптивного дизайна](/docs/responsive-design).
