## Применение

Используйте утилиты `isolate` и `isolation-auto`, чтобы указать, должен ли элемент явно создавать новый контекст стекирования.

```html
<div class="isolate ...">
  <!-- ... -->
</div>
```

## Адаптивность

Чтобы управлять свойством изоляции в определенной контрольной точке, добавьте префикс `{screen}:` к любой существующей утилите изоляции. Например, используйте `md:isolation-auto`, чтобы применить утилиту `isolation-auto` только на экранах среднего размера и выше.

```html
<div class="isolate **md:isolation-auto** ...">
  <!-- ... -->
</div>
```

Для получения дополнительной информации о функциях адаптивного дизайна Tailwind ознакомьтесь с документацией [Адаптивного дизайна](/docs/responsive-design).
