## Применение

Используйте утилиты `mix-blend-{mode}` для управления тем, как содержимое элемента должно смешиваться с фоном.

```html emerald
<template preview>
  <div class="flex justify-center">
    <div class="flex items-center justify-center rounded-md bg-emerald-300 h-24 w-48">
      <div class="mix-blend-multiply rounded-md h-32 w-32 bg-amber-300"></div>
    </div>
  </div>
</template>

<div class="bg-green-300 ...">
  <div class="**mix-blend-multiply** bg-amber-300 ..."></div>
</div>
```

## Адаптивность

Чтобы управлять свойством `mix-blend-mode` в определенной контрольной точке, добавьте префикс `{screen}:` к любой существующей утилите `mix-blend-mode`. Например, используйте `md:mix-blend-overlay`, чтобы применить утилиту `mix-blend-overlay` только при средних размерах экрана и выше.

```html
<div class="mix-blend-multiply **md:mix-blend-overlay** ...">
  <!-- ... -->
</div>
```

Для получения дополнительной информации о функциях адаптивного дизайна Tailwind ознакомьтесь с документацией [Адаптивного дизайна](/docs/responsive-design).
