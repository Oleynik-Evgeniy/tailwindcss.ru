## Применение

Используйте утилиты `grid-cols-{n}` для создания сеток с _n_ столбцами одинакового размера.

```html fuchsia
<template preview class="bg-white p-8">
  <div class="grid grid-cols-3 gap-4">
    <div class="bg-fuchsia-500 h-12 rounded-md flex items-center justify-center text-white text-2xl font-extrabold">1</div>
    <div class="bg-fuchsia-500 h-12 rounded-md flex items-center justify-center text-white text-2xl font-extrabold">2</div>
    <div class="bg-fuchsia-500 h-12 rounded-md flex items-center justify-center text-white text-2xl font-extrabold">3</div>
    <div class="bg-fuchsia-500 h-12 rounded-md flex items-center justify-center text-white text-2xl font-extrabold">4</div>
    <div class="bg-fuchsia-500 h-12 rounded-md flex items-center justify-center text-white text-2xl font-extrabold">5</div>
    <div class="bg-fuchsia-500 h-12 rounded-md flex items-center justify-center text-white text-2xl font-extrabold">6</div>
    <div class="bg-fuchsia-500 h-12 rounded-md flex items-center justify-center text-white text-2xl font-extrabold">7</div>
    <div class="bg-fuchsia-500 h-12 rounded-md flex items-center justify-center text-white text-2xl font-extrabold">8</div>
    <div class="bg-fuchsia-500 h-12 rounded-md flex items-center justify-center text-white text-2xl font-extrabold">9</div>
  </div>
</template>

<div class="grid **grid-cols-3** gap-4">
  <div>1</div>
  <!-- ... -->
  <div>9</div>
</div>
```

## Адаптивность

Чтобы управлять столбцами сетки в определенной контрольной точке, добавьте префикс `{screen}:` к любой существующей утилите grid-template-columns. Например, используйте `md:grid-cols-6`, чтобы применить утилиту `grid-cols-6` только при средних размерах экрана и выше.

```html
<div class="grid grid-cols-1 **md:grid-cols-6**">
  <!-- ... -->
</div>
```

Для получения дополнительной информации о функциях адаптивного дизайна Tailwind ознакомьтесь с документацией [Адаптивного дизайна](/docs/responsive-design).

## Кастомизация

По умолчанию Tailwind включает утилиты grid-template-column для создания базовых сеток с 12 столбцами одинаковой ширины. Вы изменяете, добавляете или удаляете их, настраивая раздел `gridTemplateColumns` в конфигурации Вашей темы Tailwind.

Здесь у Вас есть прямой доступ к CSS свойству `grid-template-columns`, так что Вы можете сделать свои собственные значения столбцов как общие, так и сложные и зависящие от сайта, как Вам нравится.

```diff-js
  // tailwind.config.js
  module.exports = {
    theme: {
      extend: {
        gridTemplateColumns: {
          // Простая сетка из 16 столбцов
+         '16': 'repeat(16, minmax(0, 1fr))',

          // Сложная конфигурация столбца для конкретного сайта
+         'footer': '200px minmax(900px, 1fr) 100px',
        }
      }
    }
  }
```

Дополнительные сведения о настройке темы по умолчанию смотрите в [документации по настройке темы](/docs/theme#customizing-the-default-theme).
