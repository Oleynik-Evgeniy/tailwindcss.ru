import utilities from 'utilities?plugin=ringColor'

export const classes = {
  utilities,
  transformProperties: ({ selector, properties }) => {
    delete properties['--tw-ring-opacity']
    Object.keys(properties).forEach(key => {
      properties[key] = properties[key].replace(' / var(--tw-ring-opacity)', '')
    })
    return properties
  }
}

## Основы использования

### Установка цвета кольца

Используйте утилиты `ring-{color}` для установки цвета [контура кольца](/docs/ring-width).

<Example>
  <div class="flex justify-center max-w-md mx-auto gap-4 text-white text-sm text-center font-bold leading-6 text-white">
    <button class="px-4 py-2 font-semibold text-sm bg-white text-gray-700 border border-gray-300 rounded-md shadow-sm ring-2 ring-offset-2 ring-offset-gray-50 ring-blue-500">Создать аккаунт</button>
  </div>
</Example>

```html
<button class="... ring-2 **ring-blue-500**">
  Создать аккаунт
</button>
```

### Изменение непрозрачности

Управляйте непрозрачностью цвета фона элемента с помощью модификатора непрозрачности цвета.

<Example>
  <div class="flex justify-center max-w-md mx-auto gap-4 text-white text-sm text-center font-bold leading-6 text-white">
    <button class="px-4 py-2 font-semibold text-sm bg-white text-gray-700 border border-gray-300 rounded-md shadow-sm ring-2 ring-offset-2 ring-offset-gray-50 ring-blue-500/50">Создать аккаунт</button>
  </div>
</Example>

```html
<button class="... ring-2 ring-blue-500**/50**">Подписаться</button>
```

Вы можете использовать любое значение, определенное в вашей [шкале непрозрачности](/docs/opacity), или использовать произвольные значения, если вам нужно отклониться от маркеров вашего дизайна.

```html
<button class="... ring-2 ring-blue-500**/[.55]**">Подписаться</button>
```
