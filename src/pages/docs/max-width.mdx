## Применение

Установите максимальную ширину элемента с помощью утилит `max-w-{size}`.

```html emerald resizable
<template preview>
  <div class="max-w-md mx-auto rounded-md h-16 bg-emerald-500 select-none"></div>
</template>

<div class="**max-w-md** mx-auto ...">
  <!-- ... -->
</div>
```

## Ширина чтения

Утилита `max-w-prose` дает элементу максимальную ширину, оптимизированную для удобочитаемости, и адаптируется в зависимости от размера шрифта.

```html purple
<template preview>
  <div class="max-w-prose text-sm text-purple-700 p-6">
    <p>Dolore iste eaque molestias. Eius iure ut eaque accusantium. Voluptas repellendus nobis. Saepe nam accusantium magni veniam qui enim mollitia excepturi sapiente.</p>
  </div>
  <div class="max-w-prose text-base text-purple-700 p-6">
    <p>Dolore iste eaque molestias. Eius iure ut eaque accusantium. Voluptas repellendus nobis. Saepe nam accusantium magni veniam qui enim mollitia excepturi sapiente.</p>
  </div>
  <div class="max-w-prose text-xl text-purple-700 p-6">
    <p>Dolore iste eaque molestias. Eius iure ut eaque accusantium. Voluptas repellendus nobis. Saepe nam accusantium magni veniam qui enim mollitia excepturi sapiente.</p>
  </div>
</template>
```

## Ограничение ваших контрольных точек

Классы `max-w-screen-{breakpoint}` могут использоваться, чтобы дать элементу максимальную ширину, соответствующую определенной точке останова. Эти значения автоматически берутся из [раздела `theme.screens`](/docs/theme#screens) вашего файла `tailwind.config.js`.

```html
<div class="max-w-screen-2xl">
  <!-- ... -->
</div>
```

---

## Адаптивность

Чтобы контролировать максимальную ширину элемента в определенной контрольной точке, добавьте префикс `{screen}:` к любой существующей утилите максимальной ширины.

```html
<div class="max-w-sm **md:max-w-lg** ...">
  <!-- ... -->
</div>
```

Для получения дополнительной информации о функциях адаптивного дизайна Tailwind ознакомьтесь с документацией [Адаптивного дизайна](/docs/responsive-design).

## Кастомизация

### Шкала максимальной ширины

Настройте шкалу максимальной ширины Tailwind по умолчанию для классов `max-w-*` в разделе `theme.maxWidth` вашего файла `tailwind.config.js`.

```diff-js
  // tailwind.config.js
  module.exports = {
    theme: {
      maxWidth: {
+       '1/4': '25%',
+       '1/2': '50%',
+       '3/4': '75%',
      }
    }
  }
```

Классы `max-w-screen-*` происходят из [раздела `theme.screens`](/docs/theme#screens) вашего файла `tailwind.config.js`.

Дополнительные сведения о настройке темы по умолчанию смотрите в [документации по настройке темы](/docs/theme#customizing-the-default-theme).
