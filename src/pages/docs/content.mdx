## Основы использования

### Установка содержимого псевдоэлемента

Используйте утилиты `content-{value}` вместе с модификаторами `before` и `after` для установки содержимого псевдоэлементов `::before` и `::after`.

По умолчанию, `content-none` единственная доступная утилита с предварительно сконфигурированным содержимым. И хотя вы можете добавить дополнительные утилиты, [настроив свою тему](#customizing-your-theme), обычно имеет смысл просто использовать произвольное значение.

Используйте обозначение квадратных скобок, чтобы определять любое произвольное значение содержимого на лету.

<Example p="none">
  <div class="px-4 sm:px-0">
    <div class="p-8 w-full max-w-xl mx-auto bg-white shadow-lg text-gray-500">
      Более высокое разрешение означает больше, чем просто изображение лучшего качества. Благодаря дисплею Retina 6K, <a href="https://www.apple.com/pro-display-xdr/" class="text-blue-600 font-medium after:content-['_↗'] after:text-sm after:font-bold" target="_blank">Pro Display XDR</a> дает вам почти на 40 процентов больше площади экрана, чем дисплей 5K.
    </div>
  </div>
</Example>

### Ссылка на значение атрибута

Эти утилиты контента даже поддерживают функции CSS, такие как функция `attr()`, которую вы можете использовать для ссылки на значение, хранящееся в атрибуте:

<Example p="none">
  <div class="px-4 sm:px-0">
    <div class="font-semibold text-gray-900 p-8 w-full max-w-xl mx-auto bg-white shadow-lg flex items-center justify-center">
      <div before="Hello World" class="before:content-[attr(before)]"></div>
    </div>
  </div>
</Example>

```html
<div before="Hello World" class="before:content-[attr(before)]">
  <!-- ... -->
</div>
```

### Использование пробелов и подчеркиваний

Поскольку пробел обозначает конец класса в HTML, замените любые пробелы в произвольном значении подчеркиванием:

<Example p="none">
  <div class="px-4 sm:px-0">
    <div class="font-semibold text-gray-900 p-8 w-full max-w-xl mx-auto bg-white shadow-lg flex items-center justify-center">
      <div class="before:content-['Hello_World']"></div>
    </div>
  </div>
</Example>

```html
<div class="before:content-['Hello_World']">
  <!-- ... -->
</div>
```

Если вам нужно включить фактический символ подчеркивания, вы можете сделать это, экранировав его с помощью обратной косой черты:

<Example p="none">
  <div class="px-4 sm:px-0">
    <div class="font-semibold text-gray-900 p-8 w-full max-w-xl mx-auto bg-white shadow-lg flex items-center justify-center">
      <div class="before:content-['Hello\_World']"></div>
    </div>
  </div>
</Example>

```html
<div class="before:content-['Hello\_World']">
  <!-- ... -->
</div>
```

## Использование пользовательских значений

### Настройка вашей темы

По умолчанию Tailwind предоставляет только утилиту `content-none`. Вы можете настроить эти значения, отредактировав `theme.content` или `theme.extend.content` в вашем файле `tailwind.config.js`.

```diff-js tailwind.config.js
  module.exports = {
    theme: {
      extend: {
+       content: {
+         'link': 'url("/icons/link.svg")',
+       },
      }
    }
  }
```

Дополнительные сведения о настройке темы по умолчанию смотрите в документации [настройка темы](/docs/theme#customizing-the-default-theme).
