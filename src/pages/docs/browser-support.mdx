---
title: Поддержка браузера
description: Понимание, какие браузеры поддерживает Tailwind и как управлять префиксами поставщиков.
---

Начиная с версии 2.0, Tailwind CSS разработан для последних стабильных версий Chrome, Firefox, Edge и Safari и протестирован в них. Tailwind CSS v2.0 не поддерживает никакие версии IE, включая IE 11.

Если вам нужна поддержка IE 11, мы рекомендуем использовать Tailwind CSS v1.9, который по-прежнему остается отличным и очень продуктивным фреймворком.

Некоторые функции, такие как `focus-visible`, включены, но не поддерживаются всеми современными браузерами. Мы включили информацию о рекомендуемых полифилах для этих функций непосредственно вместе с документацией для самих функций.

## Префиксы поставщиков

Если вы компилируете свой CSS с помощью инструмента CLI `tailwindcss`, префиксы поставщиков будут добавлены автоматически.

Если нет, мы рекомендуем использовать [Autoprefixer](https://github.com/postcss/autoprefixer).

Чтобы использовать его, установите его через npm:

```shell
# Использование npm
npm install autoprefixer

# Использование Yarn
yarn add autoprefixer
```

Затем добавьте его в самый конец списка плагинов в конфигурации PostCSS:

```js
module.exports = {
  plugins: [
    require('tailwindcss'),
    require('autoprefixer'),
  ]
}
```
