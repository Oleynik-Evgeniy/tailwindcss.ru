## Курсив

Используйте утилиту `italic`, чтобы сделать текст курсивом.

```html emerald
<template preview>
  <p class="italic text-xl text-emerald-800">Быстрая, коричневая лиса, перепрыгнула через ленивого пса.</p>
</template>

<p class="**italic** ...">Быстрая коричневая лиса...</p>
```

## Отменить курсив

Используйте утилиту `not-italic` для обычного отображения текста. Обычно это используется для сброса курсивного текста в разных контрольных точках.

```html amber
<template preview>
  <p class="not-italic text-xl text-amber-800">Быстрая, коричневая лиса, перепрыгнула через ленивого пса.</p>
</template>

<p class="**not-italic** ...">Быстрая коричневая лиса...</p>
```

## Адаптивность

Чтобы управлять стилем шрифта элемента в определенной контрольной точке, добавьте префикс `{screen}:` к любой существующей утилите стилей шрифтов. Например, используйте `md:not-italic`, чтобы применить утилиту `not-italic` только при средних размерах экрана и выше.

```html
<p class="italic **md:not-italic** ...">
  Быстрая, коричневая лиса, перепрыгнула через ленивого пса.
</p>
```

Для получения дополнительной информации о функциях адаптивного дизайна Tailwind ознакомьтесь с документацией [Адаптивного дизайна](/docs/responsive-design).
