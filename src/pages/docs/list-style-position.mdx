## Основы использования

### Установка позиции стиля списка

Управляйте положением маркеров и отступом текста в списке с помощью утилит `list-inside` и `list-outside`.

<Example>
  <div class="flex flex-col sm:flex-row gap-8">
    <div class="relative">
      <div class="absolute left-8 -top-8 -bottom-8 w-px bg-pink-400/40"></div>
      <p class="font-medium text-sm text-gray-500 font-mono mb-3 ml-8">list-inside</p>
      <ul class="list-disc list-inside text-normal text-gray-700 bg-white rounded-xl shadow-lg ring-1 ring-gray-900/5 p-4 pl-8">
        <li>5 чашек нарезанных белых грибов</li>
        <li>1/2 стакана оливкового масла</li>
        <li>3 фунта сельдерея</li>
      </ul>
    </div>
    <div class="relative">
      <div class="absolute left-8 -top-0 sm:-top-8 -bottom-8 w-px bg-pink-400/40"></div>
      <p class="font-medium text-sm text-gray-500 font-mono mb-3 ml-8">list-outside</p>
      <ul class="list-disc list-outside text-normal text-gray-700 bg-white rounded-xl shadow-lg ring-1 ring-gray-900/5 p-4 pl-8">
        <li>5 чашек нарезанных белых грибов</li>
        <li>1/2 стакана оливкового масла</li>
        <li>3 фунта сельдерея</li>
      </ul>
    </div>
  </div>
</Example>

```html
<ul class="list-inside ...">
  <li>5 чашек нарезанных белых грибов</li>
  <!-- ... -->
</ul>

<ul class="list-outside ...">
  <li>5 чашек нарезанных белых грибов</li>
  <!-- ... -->
</ul>
```
