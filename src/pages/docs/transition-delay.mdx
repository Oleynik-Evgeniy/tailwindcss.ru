## Применение

Используйте утилиты `delay-{amount}` для управления задержкой перехода элемента.

```html amber
<template preview>
  <div class="flex justify-around items-center">
    <button class="transition delay-150 duration-300 ease-in-out transform hover:scale-125 bg-amber-600 text-white font-semibold px-6 py-3 rounded-md">
      Наведи на меня
    </button>
    <button class="transition delay-300 duration-300 ease-in-out transform hover:scale-125 bg-amber-600 text-white font-semibold px-6 py-3 rounded-md">
      Наведи на меня
    </button>
    <button class="transition delay-700 duration-300 ease-in-out transform hover:scale-125 bg-amber-600 text-white font-semibold px-6 py-3 rounded-md">
      Наведи на меня
    </button>
  </div>
</template>

<button class="transition **delay-150** duration-300 ease-in-out ...">Наведи на меня</button>
<button class="transition **delay-300** duration-300 ease-in-out ...">Наведи на меня</button>
<button class="transition **delay-700** duration-300 ease-in-out ...">Наведи на меня</button>
```

## Адаптивность

Чтобы управлять задержкой перехода элемента в определенной контрольной точке, добавьте префикс `{screen}:` к любой существующей утилите задержки перехода. Например, используйте `md:delay-500`, чтобы применить утилиту `delay-500` только при средних размерах экрана и выше.

Для получения дополнительной информации о функциях адаптивного дизайна Tailwind ознакомьтесь с документацией [Адаптивного дизайна](/docs/responsive-design).

## Кастомизация

### Значения задержки

По умолчанию Tailwind предоставляет восемь универсальных утилит задержки перехода. Вы изменяете, добавляете или удаляете их, настраивая раздел `transitionDelay` в конфигурации вашей темы Tailwind.

```diff-js
  // tailwind.config.js
  module.exports = {
    theme: {
      extend: {
        transitionDelay: {
+         '0': '0ms',
+         '2000': '2000ms',
        }
      }
    }
  }
```

Дополнительные сведения о настройке темы по умолчанию смотрите в [документации по настройке темы](/docs/theme#customizing-the-default-theme).
