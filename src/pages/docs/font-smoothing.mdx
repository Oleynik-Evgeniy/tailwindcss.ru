## Сглаживание Subpixel

Используйте утилиту `subpixel-antialiased` для рендеринга текста с использованием субпиксельного сглаживания.

```html lightBlue
<template preview>
  <p class="subpixel-antialiased text-lg text-light-blue-800">Быстрая, коричневая лиса, перепрыгнула через ленивого пса.</p>
</template>

<p class="**subpixel-antialiased** ...">Быстрая коричневая лиса...</p>
```

## Сглаживание Grayscale

Используйте утилиту `antialiased` для рендеринга текста с использованием сглаживания оттенков серого.

```html emerald
<template preview>
  <p class="antialiased text-lg text-emerald-800">Быстрая, коричневая лиса, перепрыгнула через ленивого пса.</p>
</template>

<p class="**antialiased** ...">Быстрая коричневая лиса...</p>
```

## Адаптивность

Чтобы управлять сглаживанием шрифта элемента в определенной контрольной точке, добавьте префикс `{screen}:` к любой существующей утилите сглаживания шрифтов. Например, используйте `md:antialiased`, чтобы применить утилиту `antialiased` только при средних размерах экрана и выше.

```html
<p class="antialiased sm:subpixel-antialiased **md:antialiased** ...">
  Быстрая, коричневая лиса, перепрыгнула через ленивого пса.
</p>
```

Для получения дополнительной информации о функциях адаптивного дизайна Tailwind ознакомьтесь с документацией [Адаптивного дизайна](/docs/responsive-design).
