## Основы использования

### Setting the outline offset

Use the `outline-{offset}` utilities to change the offset of an element's outline.

<Example>
  <div class="grid grid-cols-1 gap-8 sm:grid-cols-3 sm:gap-4 font-bold text-white text-sm text-center font-bold">
    <div class="flex flex-col items-center shrink-0">
      <p class="font-medium text-sm text-gray-500 font-mono text-center mb-3">outline-offset-0</p>
      <button class="px-4 py-2 font-semibold text-sm bg-white text-gray-700 border border-gray-300 rounded-md shadow-sm outline outline-2 outline-offset-0 outline-sky-500">Button A</button>
    </div>
    <div class="flex flex-col items-center shrink-0">
      <p class="font-medium text-sm text-gray-500 font-mono text-center mb-3">outline-offset-2</p>
      <button class="px-4 py-2 font-semibold text-sm bg-white text-gray-700 border border-gray-300 rounded-md shadow-sm outline outline-2 outline-offset-2 outline-sky-500">Button B</button>
    </div>
    <div class="flex flex-col items-center shrink-0">
      <p class="font-medium text-sm text-gray-500 font-mono text-center mb-3">outline-offset-4</p>
      <button class="px-4 py-2 font-semibold text-sm bg-white text-gray-700 border border-gray-300 rounded-md shadow-sm outline outline-2 outline-offset-4 outline-sky-500">Button C</button>
    </div>
  </div>
</Example>

```html
<button class="outline **outline-offset-0** ...">Button A</button>
<button class="outline **outline-offset-2** ...">Button B</button>
<button class="outline **outline-offset-4** ...">Button C</button>
```

## Использование пользовательских значений

### Настройка вашей темы

You can customize the `outline-offset-{width}` utilities by editing `theme.outlineOffset` или `theme.extend.outlineOffset` в вашем файле `tailwind.config.js`.

```diff-js tailwind.config.js
  module.exports = {
    theme: {
      extend: {
+       outlineOffset: {
+         3: '3px',
+       }
      }
    }
  }
```

Дополнительные сведения о настройке темы по умолчанию смотрите в документации [настройка темы](/docs/theme#customizing-the-default-theme).
