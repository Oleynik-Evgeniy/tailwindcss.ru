## Применение

Используйте утилиты `decoration-slice` и `decoration-clone`, чтобы управлять отображением таких свойств, как `background`, `border`, `border-image`, `box-shadow`, `clip-page`, `margin` и `padding`, как если бы элемент был одним непрерывным фрагментом или отдельным блоком.

```html amber
<template preview>
  <div class="text-center text-5xl font-extrabold leading-none tracking-tight">
    <span class="decoration-clone bg-clip-text text-transparent bg-gradient-to-b from-amber-400 to-red-500">
      Привет<br>
      Мир
    </span>
  </div>
</template>

<span class="**decoration-clone** bg-gradient-to-b from-yellow-400 to-red-500 text-transparent ...">
  Привет<br>
  Мир
</span>
```

## Адаптивность

Чтобы управлять свойством `box-decoration-break` в определенной контрольной точке, добавьте префикс `{screen}:` к любой существующей утилите `box-decoration-break`. Например, используйте `md:decoration-slice`, чтобы применить утилиту `decoration-slice` только для средних размеров экрана и выше.

```html
<div class="decoration-clone **md:decoration-slice** ...">
  <!-- ... -->
</div>
```

Для получения дополнительной информации о функциях адаптивного дизайна Tailwind ознакомьтесь с документацией [Адаптивного дизайна](/docs/responsive-design).
