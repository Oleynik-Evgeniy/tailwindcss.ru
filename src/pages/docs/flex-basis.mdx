## Основы использования

### Setting the flex basis

Use the `basis-{size}` utilities to set the initial size of flex items.

<Example>
  <div class="flex flex-row space-x-4 font-mono text-white text-sm font-bold leading-6">
    <div class="basis-1/4 h-14 rounded-lg flex items-center justify-center bg-fuchsia-500 shadow-lg">01</div>
    <div class="basis-1/4 h-14 rounded-lg flex items-center justify-center bg-fuchsia-500 shadow-lg">02</div>
    <div class="basis-1/2 h-14 rounded-lg flex items-center justify-center bg-fuchsia-500 shadow-lg">03</div>
  </div>
</Example>

```html
<div class="flex flex-row">
  <div class="**basis-1/4**">01</div>
  <div class="**basis-1/4**">02</div>
  <div class="**basis-1/2**">03</div>
</div>
```

## Использование пользовательских значений

### Настройка вашей темы

The default flex basis scale is a combination of the [default spacing scale](/docs/customizing-spacing#default-spacing-scale) as well as a set of percentage based values.

Вы можете настроить масштаб интервалов, отредактировав `theme.spacing` или `theme.extend.spacing` в вашем файле `tailwind.config.js`.

```diff-js tailwind.config.js
  module.exports = {
    theme: {
      extend: {
+       spacing: {
+         '112': '28rem',
+         '128': '32rem',
+       }
      }
    }
  }
```

Alternatively, you can customize just the flex basis scale by editing `theme.flexBasis` или `theme.extend.flexBasis` в вашем файле `tailwind.config.js`.

```diff-js tailwind.config.js
  module.exports = {
    theme: {
      extend: {
+       flexBasis: {
+         '1/7': '14.2857143%',
+         '2/7': '28.5714286%',
+         '3/7': '42.8571429%',
+         '4/7': '57.1428571%',
+         '5/7': '71.4285714%',
+         '6/7': '85.7142857%',
+       }
      }
    }
  }
```

Дополнительные сведения о настройке темы по умолчанию смотрите в документации [настройка темы](/docs/theme#customizing-the-default-theme).
