## Основы использования

### Блочные и Строчные

Используйте `inline`, `inline-block` и `block` для управления потоком текста и элементов.

<Example p="none">
  <div class="mx-auto max-w-xs bg-white shadow-xl p-4 text-gray-500 text-sm leading-6 sm:text-base sm:leading-7">
    При управлении потоком текста с помощью свойства CSS <span class="inline bg-sky-100 font-bold text-sm text-gray-900 font-mono rounded">display: inline</span>
    вызовет текст внутри элемента для нормального обертывания.
    <br><br>
    При использовании свойства
    <span class="inline-block bg-sky-100 font-bold text-sm text-gray-900 font-mono rounded">display: inline-block</span>
    обернет элемент, чтобы предотвратить текст внутри не выходит за пределы своего родителя.
    <br><br>
    Наконец, использование свойства
    <span class="block bg-sky-100 font-bold text-sm text-gray-900 font-mono rounded">display: block</span>
    поместит элемент на его собственную строку и заполнит его родитель.
  </div>
</Example>

```html
<div>
  При управлении потоком текста с помощью свойства CSS
  <span class="**inline**">display: inline</span>
  вызовет текст внутри элемента для нормального обертывания.

  При использовании свойства <span class="**inline-block**">display: inline-block</span>
  обернет элемент, чтобы предотвратить текст внутри не выходит за пределы своего родителя.

  Наконец, использование свойства <span class="**block**">display: block</span>
  поместит элемент на его собственную строку и заполнит его родитель.
</div>
```

### Корень потока

Используйте `flow-root` для создания элемента блочного уровня с его собственным [контекстом форматирования блока](https://developer.mozilla.org/en-US/docs/Web/Guide/CSS/Block_formatting_context).

<Example p="none">
  <div class="mx-auto max-w-xs bg-white shadow-xl p-4 text-gray-500 text-sm leading-6 sm:text-base sm:leading-7">
    <div class="flow-root bg-stripes-purple mb-0.5">
      <div class="my-4 bg-white">Что ж, позволь мне кое-что сказать, веселый мальчик. Вы знаете эту маленькую марку, на которой написано «Публичная библиотека Нью-Йорка»? Что ж, это может ничего не значить для вас, но для меня это много значит. Чертовски много.</div>
    </div>
    <div class="flow-root bg-stripes-purple mt-0.5">
      <div class="my-4 bg-white">Конечно, смейтесь, если хотите. Я видел ваш типаж раньше: яркий, создающий сцену, щеголяющий условностями. Да, я знаю, о чем ты думаешь. Что этот парень так сильно портит старые библиотечные книги? Что ж, позвольте мне дать вам подсказку, младший.</div>
    </div>
  </div>
</Example>

```html
<div class="p-4">
  <div class="**flow-root** ...">
    <div class="my-4 ...">Что ж, позволь мне кое-что тебе сказать, ...</div>
  </div>
  <div class="**flow-root** ...">
    <div class="my-4 ...">Конечно, давай, смейся, если хочешь...</div>
  </div>
</div>
```

### Содержимое

Используйте `contents` для создания «фантомного» контейнера, дочерние элементы которого действуют как прямые дочерние элементы родителя.

<Example>
  <div class="flex gap-4 font-mono text-white text-sm font-bold leading-6 bg-stripes-purple rounded-lg text-center">
    <div class="flex-1 p-4 bg-purple-500 shadow-lg rounded-lg">01</div>
    <div class="contents">
      <div class="flex-1 p-4 bg-purple-500 shadow-lg rounded-lg">02</div>
      <div class="flex-1 p-4 bg-purple-500 shadow-lg rounded-lg">03</div>
    </div>
    <div class="flex-1 p-4 bg-purple-500 shadow-lg rounded-lg">04</div>
  </div>
</Example>

```html
<div class="flex ...">
  <div class="flex-1 ...">01</div>
  <div class="contents">
    <div class="flex-1 ...">02</div>
    <div class="flex-1 ...">03</div>
  </div>
  <div class="flex-1 ...">04</div>
</div>
```

### Скрытый

Используйте `hidden`, чтобы установить элемент на `display: none` и удалить его из макета страницы (сравните с `.invisible` из документации [visibility](/docs/visibility#invisible)).

<Example>
  <div class="flex gap-4 font-mono text-white text-sm font-bold leading-6 bg-stripes-purple rounded-lg text-center">
    <div class="hidden p-4 w-14 h-14 bg-purple-500 shadow-lg rounded-lg">01</div>
    <div class="p-4 w-14 h-14 bg-purple-500 shadow-lg rounded-lg">02</div>
    <div class="p-4 w-14 h-14 bg-purple-500 shadow-lg rounded-lg">03</div>
  </div>
</Example>

```html
<div class="flex ...">
  <div class="**hidden** ...">01</div>
  <div>02</div>
  <div>03</div>
</div>
```
