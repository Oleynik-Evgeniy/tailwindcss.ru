## Применение

Используйте `appearance-none`, чтобы сбросить стиль элемента, специфичный для браузера. Эта утилита часто используется при создании [компонентов пользовательских форм](/docs/examples/forms).

```html indigo
<template preview>
  <div class="max-w-md mx-auto">
    <div class="flex my-4">
      <select class="w-16">
        <option>Да</option>
        <option>Нет</option>
        <option>Возможно</option>
      </select>
      <div class="mx-4">
        Применены стили браузера по умолчанию
      </div>
    </div>
    <div class="flex my-4">
      <select class="appearance-none w-16">
        <option>Да</option>
        <option>Нет</option>
        <option>Возможно</option>
      </select>
      <div class="mx-4">
        Стили по умолчанию удалены
      </div>
    </div>
  </div>
</template>

<select>
  <option>Да</option>
  <option>Нет</option>
  <option>Возможно</option>
</select>

<select class="appearance-none">
  <option>Да</option>
  <option>Нет</option>
  <option>Возможно</option>
</select>
```

## Кастомизация

### Варианты

<Variants plugin="appearance" />

### Отключение

<Disabling plugin="appearance" />
