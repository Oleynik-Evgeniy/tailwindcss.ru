## Основы использования

### Все стороны

Используйте утилиты `border`, `.border-0`, `.border-2`, `.border-4` или `.border-8`, чтобы установить ширину границы для всех сторон элемента.

<Example>
  <div class="flex flex-col sm:flex-row items-center justify-around gap-4 text-white text-sm text-center font-bold leading-6">
    <div class="flex flex-col items-center shrink-0">
      <p class="font-medium text-sm text-gray-500 font-mono text-center mb-3">border</p>
      <div class="p-4 shadow-sm bg-white w-16 h-16 border-indigo-600 border"></div>
    </div>
    <div class="flex flex-col items-center shrink-0">
      <p class="font-medium text-sm text-gray-500 font-mono text-center mb-3">border-2</p>
      <div class="p-4 shadow-sm bg-white w-16 h-16 border-indigo-600 border-2"></div>
    </div>
    <div class="flex flex-col items-center shrink-0">
      <p class="font-medium text-sm text-gray-500 font-mono text-center mb-3">border-4</p>
      <div class="p-4 shadow-sm bg-white w-16 h-16 border-indigo-600 border-4"></div>
    </div>
    <div class="flex flex-col items-center shrink-0">
      <p class="font-medium text-sm text-gray-500 font-mono text-center mb-3">border-8</p>
      <div class="p-4 shadow-sm bg-white w-16 h-16 border-indigo-600 border-8"></div>
    </div>
  </div>
</Example>

```html
<div class="**border** border-indigo-600 ..."></div>
<div class="**border-2** border-indigo-600 ..."></div>
<div class="**border-4** border-indigo-600 ..."></div>
<div class="**border-8** border-indigo-600 ..."></div>
```

### Отдельные стороны

Используйте `border-{side}`, `.border-{side}-0`, `.border-{side}-2`, `.border-{side}-4` или `.border-{side}-8` для установки ширины границы для одной стороны элемента.

<Example>
  <div class="flex flex-col sm:flex-row items-center justify-around gap-4 text-white text-sm text-center font-bold leading-6">
    <div class="flex flex-col items-center shrink-0">
      <p class="font-medium text-sm text-gray-500 font-mono text-center mb-3">border-t-4</p>
      <div class="p-4 shadow-sm bg-white ring-1 ring-gray-900/5 w-16 h-16 border-indigo-500 border-t-4"></div>
    </div>
    <div class="flex flex-col items-center shrink-0">
      <p class="font-medium text-sm text-gray-500 font-mono text-center mb-3">border-r-4</p>
      <div class="p-4 shadow-sm bg-white ring-1 ring-gray-900/5 w-16 h-16 border-indigo-500 border-r-4"></div>
    </div>
    <div class="flex flex-col items-center shrink-0">
      <p class="font-medium text-sm text-gray-500 font-mono text-center mb-3">border-b-4</p>
      <div class="p-4 shadow-sm bg-white ring-1 ring-gray-900/5 w-16 h-16 border-indigo-500 border-b-4"></div>
    </div>
    <div class="flex flex-col items-center shrink-0">
      <p class="font-medium text-sm text-gray-500 font-mono text-center mb-3">border-l-4</p>
      <div class="p-4 shadow-sm bg-white ring-1 ring-gray-900/5 w-16 h-16 border-indigo-500 border-l-4"></div>
    </div>
  </div>
</Example>

```html
<div class="**border-t-4** border-indigo-500 ..."></div>
<div class="**border-r-4** border-indigo-500 ..."></div>
<div class="**border-b-4** border-indigo-500 ..."></div>
<div class="**border-l-4** border-indigo-500 ..."></div>
```

### Горизонтальные и вертикальные стороны

Используйте утилиты `border-{x|y}-{width}` для установки ширины границы с двух сторон элемента одновременно.

<Example>
  <div class="flex flex-col sm:flex-row items-center justify-around gap-4 text-white text-sm text-center font-bold leading-6">
    <div class="flex flex-col items-center shrink-0">
      <p class="font-medium text-sm text-gray-500 font-mono text-center mb-3">border-x-4</p>
      <div class="p-4 shadow-sm bg-white ring-1 ring-gray-900/5 w-16 h-16 border-indigo-500 border-x-4"></div>
    </div>
    <div class="flex flex-col items-center shrink-0">
      <p class="font-medium text-sm text-gray-500 font-mono text-center mb-3">border-y-4</p>
      <div class="p-4 shadow-sm bg-white ring-1 ring-gray-900/5 w-16 h-16 border-indigo-500 border-y-4"></div>
    </div>
  </div>
</Example>

```html
<div class="**border-x-4** border-indigo-500 ..."></div>
<div class="**border-y-4** border-indigo-500 ..."></div>
```

### Между элементами

Вы также можете добавить границы между дочерними элементами с помощью утилит `divide-{x/y}-{width}` и `divide-{color}`.

<Example p="none">
  <div class="flex flex-col divide-y divide-gray-200 font-mono text-white text-sm text-center font-bold leading-6 shadow-lg overflow-hidden max-w-sm mx-auto">
    <div class="p-4 text-gray-400 bg-white">01</div>
    <div class="p-4 text-gray-400 bg-white">02</div>
    <div class="p-4 text-gray-400 bg-white">03</div>
  </div>
</Example>

```html
<div class="**divide-y** divide-gray-200 ...">
  <div>01</div>
  <div>02</div>
  <div>03</div>
</div>
```

Дополнительные сведения смотрите в документации [Ширина разделения](/docs/divide-width) и [Цвет разделения](/docs/divide-color).

## Использование пользовательских значений

### Настройка вашей темы

По умолчанию Tailwind предоставляет пять утилит `border-width`, и такое же количество утилит на каждую сторону (горизонтальную, вертикальную, верхнюю, правую, нижнюю и левую). Вы изменяете, добавляете или удаляете их, редактируя раздел `theme.borderWidth` в вашей конфигурации Tailwind.

```diff-js tailwind.config.js
  module.exports = {
    theme: {
      borderWidth: {
        DEFAULT: '1px',
        '0': '0',
        '2': '2px',
+       '3': '3px',
        '4': '4px',
+       '6': '6px',
-       '8': '8px',
      }
    }
  }
```

Дополнительные сведения о настройке темы по умолчанию смотрите в документации [настройка темы](/docs/theme#customizing-the-default-theme).
