## Основы использования

### Setting the outline color

Use the `outline-{color}` utilities to change the color of an element's outline.

<Example>
  <div class="flex flex-col sm:flex-row items-center justify-center gap-8 sm:gap-16 font-bold text-sm text-white text-center font-bold">
    <div class="flex flex-col items-center shrink-0">
      <p class="font-medium text-sm text-gray-500 font-mono text-center mb-3">outline-blue-500</p>
      <button class="px-4 py-2 font-semibold text-sm bg-white text-gray-700 border border-gray-300 rounded-md shadow-sm outline outline-2 outline-offset-2 outline-blue-500">Button A</button>
    </div>
    <div class="flex flex-col items-center shrink-0">
      <p class="font-medium text-sm text-gray-500 font-mono text-center mb-3">outline-cyan-500</p>
      <button class="px-4 py-2 font-semibold text-sm bg-white text-gray-700 border border-gray-300 rounded-md shadow-sm outline outline-2 outline-offset-2 outline-cyan-500">Button B</button>
    </div>
    <div class="flex flex-col items-center shrink-0">
      <p class="font-medium text-sm text-gray-500 font-mono text-center mb-3">outline-pink-500</p>
      <button class="px-4 py-2 font-semibold text-sm bg-white text-gray-700 border border-gray-300 rounded-md shadow-sm outline outline-2 outline-offset-2 outline-pink-500">Button C</button>
    </div>
  </div>
</Example>

```html
<button class="outline outline-offset-2 **outline-blue-500** ...">Button A</button>
<button class="outline outline-offset-2 **outline-cyan-500** ...">Button B</button>
<button class="outline outline-offset-2 **outline-pink-500** ...">Button C</button>
```

### Изменение непрозрачности

Control the opacity of an element's outline color using the color opacity modifier.

<Example>
  <div class="text-center font-bold text-sm text-center font-bold text-white">
    <div class="flex flex-col items-center shrink-0">
      <p class="font-medium text-sm text-gray-500 font-mono text-center mb-3">outline-blue-500/50</p>
      <button class="px-4 py-2 font-semibold text-sm bg-white text-gray-700 border border-gray-300 rounded-md shadow-sm outline outline-2 outline-offset-2 outline-blue-500/50">Save Changes</button>
    </div>
  </div>
</Example>

```html
<button class="outline-2 outline-blue-500**/50** ...">Save Changes</button>
```

You can use any value defined in your [opacity scale](/docs/opacity), or use arbitrary values if you need to deviate from your design tokens.

```html
<button class="outline-4 outline-pink-400**/[.55]** ..."></button>
```
