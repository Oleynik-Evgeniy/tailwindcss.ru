```html lightBlue

    <div class="flex-1 rounded-md bg-light-blue-500 text-white text-2xl font-extrabold flex items-center justify-center py-3">2</div>
    <div class="flex-1 rounded-md bg-light-blue-500 text-white text-2xl font-extrabold flex items-center justify-center py-3">3</div>

    <div class="flex-1 rounded-md bg-light-blue-500 text-white text-2xl font-extrabold flex items-center justify-center py-3">6</div>
  </div>
</template>
```

## Адаптивность

Чтобы поместить контент в определенную контрольную точку, добавьте префикс `{screen}:` к любой существующей утилите `place-content`. Например, используйте `md:place-content-center`, чтобы применить утилиту `place-content-center` только при средних размерах экрана и выше.

```html
<div class="place-content-start md:place-content-center">
  <!-- ... -->
</div>
```

Для получения дополнительной информации о функциях адаптивного дизайна Tailwind ознакомьтесь с документацией [Адаптивного дизайна](/docs/responsive-design).
