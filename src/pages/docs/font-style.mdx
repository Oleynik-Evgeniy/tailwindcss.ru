## Основы использования

### Выделение текста курсивом

Утилиту `italic` можно использовать для выделения текста курсивом. Точно так же утилиту `not-italic` можно использовать для обычного отображения текста - обычно для сброса курсивного текста в разных контрольных точках.

<Example>
  <div class="flex flex-col gap-8">
    <div>
      <span class="font-medium text-sm text-gray-500 font-mono mb-3">italic</span>
      <p class="italic text-lg font-medium text-gray-900">
        Быстрая коричневая лиса прыгает через ленивую собаку.
      </p>
    </div>
    <div>
      <span class="font-medium text-sm text-gray-500 font-mono mb-3">not-italic</span>
      <p class="not-italic text-lg font-medium text-gray-900">
        Быстрая коричневая лиса прыгает через ленивую собаку.
      </p>
    </div>
  </div>
</Example>

```html
<p class="**italic** ...">Быстрая коричневая лиса ...</p>
<p class="**not-italic** ...">Быстрая коричневая лиса ...</p>
```
