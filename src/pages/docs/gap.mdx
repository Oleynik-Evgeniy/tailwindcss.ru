## Применение

Используйте `gap-{size}`, чтобы изменить зазор между строками и столбцами в макетах сетки.

```html indigo
<template preview class="p-6">
  <div>
    <div class="grid grid-cols-2 gap-4">
      <div class="h-16 rounded-md text-center bg-indigo-500"></div>
      <div class="h-16 rounded-md text-center bg-indigo-500"></div>
      <div class="h-16 rounded-md text-center bg-indigo-500"></div>
      <div class="h-16 rounded-md text-center bg-indigo-500"></div>
    </div>
  </div>
</template>

<div class="grid **gap-4** grid-cols-2">
  <div>1</div>
  <div>2</div>
  <div>3</div>
  <div>4</div>
</div>
```

### Независимое изменение промежутков между строками и столбцами

Используйте `gap-x-{size}` и `gap-y-{size}` для независимого изменения зазора между строками и столбцами.

```html rose
<template preview class="p-6">
  <div>
    <div class="grid grid-cols-3 gap-x-8 gap-y-4">
      <div class="h-16 rounded-md text-center bg-rose-500"></div>
      <div class="h-16 rounded-md text-center bg-rose-500"></div>
      <div class="h-16 rounded-md text-center bg-rose-500"></div>
      <div class="h-16 rounded-md text-center bg-rose-500"></div>
      <div class="h-16 rounded-md text-center bg-rose-500"></div>
      <div class="h-16 rounded-md text-center bg-rose-500"></div>
    </div>
  </div>
</template>

<div class="grid **gap-x-8** **gap-y-4** grid-cols-3">
  <div>1</div>
  <div>2</div>
  <div>3</div>
  <div>4</div>
  <div>5</div>
  <div>6</div>
</div>
```

## Адаптивность

Чтобы контролировать зазоры в определенной контрольной точке, добавьте префикс `{screen}:` к любой существующей утилите промежутка. Например, используйте `md:gap-6`, чтобы применить утилиту `gap-6` только при средних размерах экрана и выше.

```html
<div class="grid gap-4 **md:gap-6** ...">
  <!-- ... -->
</div>
```

Для получения дополнительной информации о функциях адаптивного дизайна Tailwind ознакомьтесь с документацией [Адаптивного дизайна](/docs/responsive-design).

## Кастомизация

По умолчанию шкала промежутка Tailwind соответствует настроенной Вами [шкале интервалов](/docs/customizing-spacing).

Вы можете настроить глобальную шкалу интервалов в разделах `theme.spacing` или `theme.extend.spacing` Вашего файла `tailwind.config.js`:

```diff-js
  // tailwind.config.js
  module.exports = {
    theme: {
      extend: {
        spacing: {
+         '72': '18rem',
+         '84': '21rem',
+         '96': '24rem',
        }
      }
    }
  }
```

Чтобы настроить масштаб промежутка отдельно, используйте раздел `gap` в конфигурации Вашей темы Tailwind.

```diff-js
  // tailwind.config.js
  module.exports = {
    theme: {
      extend: {
        gap: {
+         '11': '2.75rem',
+         '13': '3.25rem',
        }
      }
    }
  }
```

Дополнительные сведения о настройке темы по умолчанию смотрите в [документации по настройке темы](/docs/theme#customizing-the-default-theme).
