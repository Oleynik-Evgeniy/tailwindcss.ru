## Адаптивность

Чтобы управлять свойством overscroll-behavior в определенной контрольной точке, добавьте префикс `{screen}:` к любой существующей утилите overscroll-behavior. Например, используйте `md:overscroll-contain`, чтобы применить утилиту `overscroll-contain` только для средних размеров экрана и выше.

```html
<div class="overscroll-auto **md:overscroll-contain lg:overscroll-none** ...">
  <!-- ... -->
</div>
```

Для получения дополнительной информации о функциях адаптивного дизайна Tailwind ознакомьтесь с документацией [Адаптивного дизайна](/docs/responsive-design).
