## Исходный

Используйте `flex-initial`, чтобы позволить flex-элементу сжиматься, но не увеличиваться, учитывая его начальный размер:

```html indigo

  <p class="text-sm text-indigo-600">Элементы не расширяются, когда есть лишнее место</p>
  <div class="mt-2 flex space-x-4">
    <div class="flex-initial rounded-md bg-indigo-500 text-white font-semibold flex items-center justify-center py-3 px-6">
      Короткий
    </div>
    <div class="flex-initial rounded-md bg-indigo-500 text-white font-semibold flex items-center justify-center py-3 px-6">
      Средняя длина
    </div>
  </div>

  <p class="mt-8 text-sm text-indigo-600">При необходимости элементы сжимаются, если это возможно</p>
  <div class="mt-2 flex space-x-4">
    <div class="flex-initial rounded-md bg-indigo-500 text-white font-semibold flex items-center justify-center py-3 px-6">
      Короткий
    </div>
    <div class="flex-initial rounded-md bg-indigo-500 text-white font-semibold flex items-center justify-center py-3 px-6">

  </div>

<div class="flex">
```

## Гибкий 1

Используйте `flex-1`, чтобы позволить flex-элементу увеличиваться и уменьшаться по мере необходимости, игнорируя его начальный размер:

```html fuchsia
<template preview>
  <p class="text-sm text-fuchsia-600">Поведение по умолчанию</p>
  <div class="mt-2 flex space-x-4">
    <div class="rounded-md bg-fuchsia-500 text-white font-semibold flex items-center justify-center py-3 px-6">
      Короткий
    </div>
    <div class="rounded-md bg-fuchsia-500 text-white font-semibold flex items-center justify-center py-3 px-6">
      Средняя длина
    </div>
    <div class="rounded-md bg-fuchsia-500 text-white font-semibold flex items-center justify-center py-3 px-6">
      Значительно больший объем контента
    </div>
  </div>

  <p class="mt-8 text-sm text-fuchsia-600">С <code>.flex-1</code></p>
  <div class="mt-2 flex space-x-4">
    <div class="flex-1 rounded-md bg-fuchsia-500 text-white font-semibold flex items-center justify-center py-3 px-6">
      Короткий
    </div>
    <div class="flex-1 rounded-md bg-fuchsia-500 text-white font-semibold flex items-center justify-center py-3 px-6">
      Средняя длина
    </div>
    <div class="flex-1 rounded-md bg-fuchsia-500 text-white font-semibold flex items-center justify-center py-3 px-6">
      Значительно больший объем контента
    </div>
  </div>
</template>

<div class="flex">
  <div class="**flex-1** ...">
    <!-- Будет увеличиваться и уменьшаться по мере необходимости без учета первоначального размера -->

  <div class="**flex-1** ...">
    <!-- Будет увеличиваться и уменьшаться по мере необходимости без учета первоначального размера -->
  </div>
  <div class="**flex-1** ...">
    <!-- Будет увеличиваться и уменьшаться по мере необходимости без учета первоначального размера -->
```

## Ничего

Используйте `flex-none`, чтобы предотвратить расширение или сжатие гибкого элемента:

```html emerald
<template preview>
  <div class="flex space-x-4">
    <div class="flex-1 rounded-md bg-emerald-300 text-white font-semibold flex items-center justify-center py-3 px-6">
      Элемент, который при необходимости может увеличиваться или уменьшаться
    </div>
    <div class="flex-none rounded-md bg-emerald-500 text-white font-semibold flex items-center justify-center py-3 px-6">
      Элемент, который не может увеличиваться или уменьшаться
    </div>
    <div class="flex-1 rounded-md bg-emerald-300 text-white font-semibold flex items-center justify-center py-3 px-6">
      Элемент, который при необходимости может увеличиваться или уменьшаться
    </div>
  </div>
</template>

<div class="flex ...">
  <div class="flex-1 ...">
    <!-- Будет расширяться и уменьшаться по мере необходимости -->
  </div>
  <div class="**flex-none** ...">
    <!-- Не будет расширяться и сжиматься -->
  </div>
  <div class="flex-1 ...">
    <!-- Будет расширяться и уменьшаться по мере необходимости -->
  </div>
</div>
```

## Адаптивность

Чтобы контролировать увеличение и уменьшение flex-элемента в определенной контрольной точке, добавьте префикс `{screen}:` к любому существующему классу утилиты. Например, используйте `md:flex-1`, чтобы применить утилиту `flex-1` только при средних размерах экрана и выше.

```html
<div class="flex ...">
  <!-- ... -->
  <div class="flex-none **md:flex-1** ...">
    Адаптивный гибкий элемент
  </div>
  <!-- ... -->
</div>
```

Для получения дополнительной информации о функциях адаптивного дизайна Tailwind ознакомьтесь с документацией [Адаптивного дизайна](/docs/responsive-design).

## Кастомизация

### Гибкие значения

По умолчанию Tailwind предоставляет четыре `flex` утилиты. Вы изменяете, добавляете или удаляете их, редактируя раздел `theme.flex` Вашей конфигурации Tailwind.

```diff-js
  // tailwind.config.js
  module.exports = {
    theme: {
      flex: {
        '1': '1 1 0%',
        auto: '1 1 auto',
-       initial: '0 1 auto',
+       inherit: 'inherit',
        none: 'none',
+       '2': '2 2 0%',
      }
    }
  }
```
