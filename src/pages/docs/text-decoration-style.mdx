## Основы использования

### Setting the text decoration style

Use the `decoration-{style}` utilities to change the style of an element's [text decoration](/docs/text-decoration).

<Example>
  <div class="flex flex-col gap-8 text-gray-900">
    <div>
      <div class="font-medium text-sm text-gray-500 font-mono mb-3">decoration-solid</div>
      <p class="text-lg font-medium underline decoration-gray-900 decoration-solid">Быстрая коричневая лиса прыгает через ленивую собаку.</p>
    </div>
    <div>
      <div class="font-medium text-sm text-gray-500 font-mono mb-3">decoration-double</div>
      <p class="text-lg font-medium underline decoration-gray-900 decoration-double">Быстрая коричневая лиса прыгает через ленивую собаку.</p>
    </div>
    <div>
      <div class="font-medium text-sm text-gray-500 font-mono mb-3">decoration-dotted</div>
      <p class="text-lg font-medium underline decoration-gray-900 decoration-dotted">Быстрая коричневая лиса прыгает через ленивую собаку.</p>
    </div>
    <div>
      <div class="font-medium text-sm text-gray-500 font-mono mb-3">decoration-dashed</div>
      <p class="text-lg font-medium underline decoration-gray-900 decoration-dashed">Быстрая коричневая лиса прыгает через ленивую собаку.</p>
    </div>
    <div>
      <div class="font-medium text-sm text-gray-500 font-mono mb-3">decoration-wavy</div>
      <p class="text-lg font-medium underline decoration-gray-900 decoration-wavy">Быстрая коричневая лиса прыгает через ленивую собаку.</p>
    </div>
  </div>
</Example>

```html
<p class="underline **decoration-solid** ...">Быстрая коричневая лиса...</p>
<p class="underline **decoration-double** ...">Быстрая коричневая лиса...</p>
<p class="underline **decoration-dotted** ...">Быстрая коричневая лиса...</p>
<p class="underline **decoration-dashed** ...">Быстрая коричневая лиса...</p>
<p class="underline **decoration-wavy** ...">Быстрая коричневая лиса...</p>
```
