## Применение

Используйте утилиты `hue-rotate-{amount}` вместе с утилитой `filter`, чтобы размыть элемент.

```html
<div class="filter hue-rotate-15 ...">
  <!-- ... -->
</div>
```

## Адаптивность

Чтобы управлять вращением оттенка элемента в определенной контрольной точке, добавьте префикс `{screen}:` к любой существующей утилите `hue-rotate`. Например, используйте `md:hue-rotate-60`, чтобы применить утилиту `hue-rotate-60` только при средних размерах экрана и выше.

```html
<div class="filter hue-rotate-15 **md:hue-rotate-60** ...">
  <!-- ... -->
</div>
```

Для получения дополнительной информации о функциях адаптивного дизайна Tailwind ознакомьтесь с документацией [Адаптивного дизайна](/docs/responsive-design).

## Кастомизация

Вы можете настроить, какие утилиты `hue-rotate` будут генерироваться, используя ключ `hueRotate` в разделе `theme` вашего файла `tailwind.config.js`.

```diff-js
  // tailwind.config.js
  module.exports = {
    theme: {
+     extend: {
+       hueRotate: {
+         '-270': '-270deg',
+         270: '270deg',
+       }
+     }
    }
  }
```

Дополнительные сведения о настройке темы по умолчанию смотрите в [документации по настройке темы](/docs/theme#customizing-the-default-theme).
