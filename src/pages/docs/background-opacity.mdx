## Применение

Управляйте непрозрачностью цвета фона элемента с помощью утилит `bg-opacity-{amount}`.

```html indigo
<template preview>
  <div class="grid grid-cols-2 gap-2 sm:grid-cols-5">
    <div class="h-16 rounded-md bg-opacity-100 bg-indigo-600 font-extrabold text-white flex justify-center items-center">100%</div>
    <div class="h-16 rounded-md bg-opacity-75 bg-indigo-600 font-extrabold text-white flex justify-center items-center">75%</div>
    <div class="h-16 rounded-md bg-opacity-50 bg-indigo-600 font-extrabold text-white flex justify-center items-center">50%</div>
    <div class="h-16 rounded-md bg-opacity-25 bg-indigo-600 font-extrabold text-white flex justify-center items-center">25%</div>
    <div class="h-16 rounded-md bg-opacity-0 bg-indigo-600 font-extrabold text-white flex justify-center items-center">0%</div>
  </div>
</template>

<div class="bg-indigo-600 **bg-opacity-100** ..."></div>
<div class="bg-indigo-600 **bg-opacity-75** ..."></div>
<div class="bg-indigo-600 **bg-opacity-50** ..."></div>
<div class="bg-indigo-600 **bg-opacity-25** ..."></div>
<div class="bg-indigo-600 **bg-opacity-0** ..."></div>
```

## Адаптивность

Чтобы управлять непрозрачностью цвета фона элемента в определенной контрольной точке, добавьте префикс `{screen}:` к любой существующей утилите прозрачности цвета фона. Например, используйте `md:bg-opacity-50`, чтобы применить утилиту `bg-opacity-50` только при средних размерах экрана и выше.

```html
<div class="bg-blue-500 bg-opacity-75 **md:bg-opacity-50**">
  <!-- ... -->
</div>
```

Для получения дополнительной информации о функциях адаптивного дизайна Tailwind ознакомьтесь с документацией [Адаптивного дизайна](/docs/responsive-design).

## Кастомизация

Чтобы настроить значения непрозрачности сразу для всех утилит, связанных с непрозрачностью, используйте раздел `opacity` в конфигурации темы `tailwind.config.js`:

```diff-js
  // tailwind.config.js
  module.exports = {
    theme: {
      extend: {
        opacity: {
+         '15': '0.15',
+         '35': '0.35',
+         '65': '0.65',
        }
      }
    }
  }
```

Если вы хотите настроить только утилиты непрозрачности фона, используйте раздел `backgroundOpacity`:

```diff-js
  // tailwind.config.js
  module.exports = {
    theme: {
      extend: {
        backgroundOpacity: {
+         '10': '0.1',
+         '20': '0.2',
+         '95': '0.95',
        }
      }
    }
  }
```

Дополнительные сведения о настройке темы по умолчанию смотрите в [документации по настройке темы](/docs/theme#customizing-the-default-theme).
