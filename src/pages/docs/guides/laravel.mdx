### Настройте Tailwind с помощью Laravel Mix

В свой `webpack.mix.js` добавьте `tailwindcss` в качестве плагина PostCSS:

```diff-js
  // webpack.mix.js
  mix.js("resources/js/app.js", "public/js")
    .postCss("resources/css/app.css", "public/css", [
+     require("tailwindcss"),
    ]);
```

```preval include
tool: Laravel
file: ./resources/css/app.css
```

Затем импортируйте свою таблицу стилей в основной макет Blade (обычно `resources/views/layouts/app.php` или аналогичный) и добавьте метатег адаптивного окна просмотра, если он еще не присутствует:

```diff-html
  <!doctype html>
  <head>
    <!-- ... --->
+   <meta charset="UTF-8" />
+   <meta name="viewport" content="width=device-width, initial-scale=1.0" />
+   <link href="{{ asset('css/app.css') }}" rel="stylesheet">
  </head>
  <!-- ... --->
```
