## Применение

Установите максимальную ширину элемента с помощью утилит `max-w-{size}`.

```html emerald
<template preview>
  <div class="max-w-md mx-auto rounded-md bg-emerald-500 text-white text-2xl font-extrabold flex items-center justify-center py-3">
    max-w-md
  </div>
</template>

<div class="**max-w-md** mx-auto ...">
  max-w-md
</div>
```

## Проза

Мы добавили новую максимальную ширину по умолчанию под названием `max-w-prose`. Если Вы знакомы с [плагином типографики](https://github.com/tailwindlabs/tailwindcss-typography), Вы можете узнать его. Идея этого класса заключается в том, что он позволяет Вам иметь оптимальную ширину чтения.

```html purple
<template preview>
  <div class="max-w-prose text-sm text-purple-700 p-6">
    <p class="font-bold">Почему вы никогда не видите слонов, прячущихся на деревьях?</p>
    <p>Потому что они так хороши в этом. Lorem ipsum dolor sit amet consectetur adipisicing elit. Quas cupiditate laboriosam fugiat.</p>
  </div>
</template>

<div class="**max-w-prose** ...">
  <p class="font-bold">Почему вы никогда не видите слонов, прячущихся на деревьях?</p>
  <p>Потому что они так хороши в этом. Lorem ipsum dolor sit amet consectetur adipisicing elit. Quas cupiditate laboriosam fugiat.</p>
</div>
```

## Адаптивность

Чтобы контролировать максимальную ширину элемента в определенной точке останова, добавьте префикс `{screen}:` к любой существующей утилите максимальной ширины.

```html
<div class="max-w-sm **md:max-w-lg** ...">
  <!-- ... -->
</div>
```

Для получения дополнительной информации о функциях адаптивного дизайна Tailwind ознакомьтесь с документацией [Адаптивного дизайна](/docs/responsive-design).

## Кастомизация

### Шкала максимальной ширины

Настройте масштаб максимальной ширины Tailwind по умолчанию в разделе `theme.maxWidth` Вашего файла `tailwind.config.js`.

```diff-js
  // tailwind.config.js
  module.exports = {
    theme: {
      maxWidth: {
+       '1/4': '25%',
+       '1/2': '50%',
+       '3/4': '75%',
      }
    }
  }
```

Дополнительные сведения о настройке темы по умолчанию смотрите в [документации по настройке темы](/docs/theme#customizing-the-default-theme).
