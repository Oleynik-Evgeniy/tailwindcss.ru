---
title: "Установите Tailwind CSS с Nuxt.js"
description: "Настройка Tailwind CSS в проекте Nuxt.js."
tool: Nuxt.js
---


```preval installation
reference:
  name: Create Nuxt App
  link: https://nuxtjs.org/guides/get-started/installation#using-create-nuxt-app
script: npx create-nuxt-app
disclaimer: >
  **Не выбирайте "Tailwind CSS" когда будет предложено выбрать "UI Framework"**.
  Это вызовет конфликты, которые не позволят вам установить Tailwind CSS v2.0 из-за модуля `@nuxtjs/tailwindcss` в зависимости от старой версии Tailwind.
```

---

```preval setup
version:
  Nuxt.js: compat-7
uninstall:
  - '@nuxtjs/tailwindcss'
  - tailwindcss
dependencies:
  - '@nuxtjs/tailwindcss'
```

Добавьте модуль `@nuxtjs/tailwindcss` в раздел `buildModules` вашего файла `nuxt.config.js`:

```js
// nuxt.config.js
export default {
  buildModules: ['@nuxtjs/tailwindcss']
}
```

```preval configuration
postcss: false
purge:
  - ./components/**/*.{vue,js}
  - ./layouts/**/*.vue
  - ./pages/**/*.vue
  - ./plugins/**/*.{js,ts}
  - ./nuxt.config.{js,ts}
```

```preval include
file: ./assets/css/tailwind.css
```

---

```preval finish
scripts:
  - npm run dev
```
