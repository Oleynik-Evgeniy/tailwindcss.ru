## Применение

Используйте утилиты `backdrop-hue-rotate-{amount}` вместе с утилитой `backdrop-filter`, чтобы размыть элемент.

```html
<div class="backdrop-filter backdrop-hue-rotate-15 ...">
  <!-- ... -->
</div>
```

## Адаптивность

Чтобы управлять вращением оттенка фона элемента в определенной контрольной точке, добавьте префикс `{screen}:` к любой существующей утилите фона `hue-rotate`. Например, используйте `md:backdrop-hue-rotate-60`, чтобы применить утилиту `backdrop-hue-rotate-60` только при средних размерах экрана и выше.

```html
<div class="backdrop-filter backdrop-hue-rotate-15 **md:backdrop-hue-rotate-60** ...">
  <!-- ... -->
</div>
```

Для получения дополнительной информации о функциях адаптивного дизайна Tailwind ознакомьтесь с документацией [Адаптивного дизайна](/docs/responsive-design).

## Кастомизация

Вы можете настроить, какие утилиты `backdrop-hue-rotate` будут генерироваться, используя ключ `backdropHueRotate` в разделе `theme` вашего файла `tailwind.config.js`.

```diff-js
  // tailwind.config.js
  module.exports = {
    theme: {
+     extend: {
+       backdropHueRotate: {
+         '-270': '-270deg',
+         270: '270deg',
+       }
+     }
    }
  }
```

Дополнительные сведения о настройке темы по умолчанию смотрите в [документации по настройке темы](/docs/theme#customizing-the-default-theme).
