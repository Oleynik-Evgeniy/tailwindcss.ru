## Применение

Используйте утилиты `stroke-{width}`, чтобы установить ширину штриха SVG.

Полезно для стилизации наборов иконок, таких как [Heroicons](http://heroicons.com/), которые полностью нарисованы штрихами.

```html emerald
<template preview>
  <div class="text-center">
    <svg class="stroke-current stroke-1 text-emerald-600 inline-block h-24 w-24" viewBox="0 0 24 24" fill="none" stroke-linecap="round" stroke-linejoin="round">
      <circle cx="12" cy="12" r="10" />
      <line x1="14.31" y1="8" x2="20.05" y2="17.94" />
      <line x1="9.69" y1="8" x2="21.17" y2="8" />
      <line x1="7.38" y1="12" x2="13.12" y2="2.06" />
      <line x1="9.69" y1="16" x2="3.95" y2="6.06" />
      <line x1="14.31" y1="16" x2="2.83" y2="16" />
      <line x1="16.62" y1="12" x2="10.88" y2="21.94" />
    </svg>
    <svg class="stroke-current stroke-2 text-emerald-600 inline-block h-24 w-24" viewBox="0 0 24 24" fill="none" stroke-linecap="round" stroke-linejoin="round">
      <circle cx="12" cy="12" r="10" />
      <line x1="14.31" y1="8" x2="20.05" y2="17.94" />
      <line x1="9.69" y1="8" x2="21.17" y2="8" />
      <line x1="7.38" y1="12" x2="13.12" y2="2.06" />
      <line x1="9.69" y1="16" x2="3.95" y2="6.06" />
      <line x1="14.31" y1="16" x2="2.83" y2="16" />
      <line x1="16.62" y1="12" x2="10.88" y2="21.94" />
    </svg>
  </div>
</template>

<svg class="stroke-current **stroke-1** text-green-600 ..."></svg>
<svg class="stroke-current **stroke-2** text-green-600 ..."></svg>
```

## Адаптивность

Чтобы управлять шириной штриха элемента SVG в определенной контрольной точке, добавьте префикс `{screen}:` к любой существующей утилите ширины. Например, добавление класса `md:stroke-2` к элементу приведет к применению утилиты `stroke-2` при средних размерах экрана и выше.

```html
<svg class="stroke-1 **md:stroke-2** ...">
  <!-- ... -->
</svg>
```

Для получения дополнительной информации о функциях адаптивного дизайна Tailwind ознакомьтесь с документацией [Адаптивного дизайна](/docs/responsive-design).

## Кастомизация

Управляйте тем, какие утилиты ширины штриха генерирует Tailwind, настраивая раздел `theme.strokeWidth` в вашем файле `tailwind.config.js`:

```diff-js
  // tailwind.config.js
  module.exports = {
    theme: {
      extend: {
        strokeWidth: {
+         '3': '3',
+         '4': '4',
        }
      }
    }
  }
```

Дополнительные сведения о настройке темы по умолчанию смотрите в [документации по настройке темы](/docs/theme#customizing-the-default-theme).
