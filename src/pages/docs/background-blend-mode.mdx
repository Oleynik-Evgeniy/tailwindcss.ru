## Применение

Используйте утилиты `bg-blend-{mode}` для управления тем, как фоновое изображение элемента должно смешиваться с цветом фона.

```html
<div class="bg-blend-multiply ...">
  <!-- ... -->
</div>
```

## Адаптивность

Чтобы управлять свойством `background-blend-mode` в определенной контрольной точке, добавьте префикс `{screen}:` к любой существующей утилите `background-blend-mode`. Например, используйте `md:bg-blend-darken`, чтобы применить утилиту `bg-blend-darken` только на средних размерах экрана и выше.

```html
<div class="bg-blend-lighten **md:bg-blend-darken** ...">
  <!-- ... -->
</div>
```

Для получения дополнительной информации о функциях адаптивного дизайна Tailwind ознакомьтесь с документацией [Адаптивного дизайна](/docs/responsive-design).
