## Добавить отступ с одной стороны

Управляйте отступом на одной стороне элемента с помощью утилит `m{t|r|b|l}-{size}`.

Например, `mt-6` добавит `1.5rem` отступ элементу сверху, `mr-4` добавит `1rem` отступ элементу справа, `mb-8` добавит `2rem` отступ элементу снизу, и `ml-2` добавит `0.5rem` отступ элементу слева.

<Example p="none">
  <div class="relative font-mono text-white text-sm font-bold leading-6 h-56">
    <div class="absolute top-0 left-1/2 -translate-x-1/2 md:-ml-24">
      <div class="flow-root bg-stripes-purple rounded-b-lg">
        <div class="bg-purple-500 rounded-lg shadow-lg p-4 mt-6">mt-6</div>
      </div>
    </div>
    <div class="absolute right-0 top-1/2 -translate-y-1/2">
      <div class="flow-root bg-stripes-purple rounded-l-lg">
        <div class="flex-none bg-purple-500 rounded-lg shadow-lg p-4 mr-4">mr-4</div>
      </div>
    </div>
    <div class="absolute bottom-0 left-1/2 -translate-x-1/2 md:ml-24">
      <div class="flow-root bg-stripes-purple rounded-t-lg">
        <div class="bg-purple-500 rounded-lg shadow-lg p-4 mb-8">mb-8</div>
      </div>
    </div>
    <div class="absolute left-0 top-1/2 -translate-y-1/2">
      <div class="flow-root bg-stripes-purple rounded-r-lg">
        <div class="flex-none bg-purple-500 rounded-lg shadow-lg p-4 ml-2">ml-2</div>
      </div>
    </div>
  </div>
</Example>

```html
<div class="**mt-6** ...">mt-6</div>
<div class="**mr-4** ...">mr-4</div>
<div class="**mb-8** ...">mb-8</div>
<div class="**ml-2** ...">ml-2</div>
```

## Добавить горизонтальный отступ

Управляйте горизонтальным отступом элемента с помощью утилит `mx-{size}`.

<Example>
  <div class="flex justify-center font-mono text-white text-sm font-bold leading-6">
    <div class="bg-stripes-indigo rounded-lg">
      <div class="bg-indigo-500 rounded-lg shadow-lg p-4 mx-8">mx-8</div>
    </div>
  </div>
</Example>

```html
<div class="**mx-8** ...">mx-8</div>
```

### Добавить вертикальный отступ

Управляйте вертикальным отступом элемента с помощью утилит `my-{size}`.

<Example>
  <div class="flex justify-center font-mono text-white text-sm font-bold leading-6">
    <div class="flow-root bg-stripes-pink rounded-lg">
      <div class="bg-pink-500 rounded-lg shadow-lg p-4 my-8">my-8</div>
    </div>
  </div>
</Example>

```html
<div class="**my-8** ...">my-8</div>
```

### Добавить отступ для всех сторон

Контролируйте отступы со всех сторон элемента с помощью утилит `m-{size}`.

<Example>
  <div class="flex justify-center font-mono text-white text-sm font-bold leading-6">
    <div class="flow-root bg-stripes-blue rounded-lg">
      <div class="bg-blue-500 rounded-lg shadow-lg p-4 m-8">m-8</div>
    </div>
  </div>
</Example>

```html
<div class="**m-8** ...">m-8</div>
```

### Использование отрицательных значений

Чтобы использовать отрицательное значение маржи, поставьте перед именем класса дефис, чтобы преобразовать его в отрицательное значение.

<Example>
  <div class="flex justify-center font-mono text-white text-sm font-bold leading-6">
    <div class="flex flex-col items-center">
      <div class="relative w-36 h-16 bg-light-blue-400/20 border border-light-blue-700/10 rounded-md overflow-hidden"></div>
      <div class="relative -mt-8 bg-light-blue-500 rounded-md flex items-center justify-center p-4 shadow-lg">-mt-8</div>
    </div>
  </div>
</Example>

```html
<div class="w-36 h-16 bg-light-blue-400 opacity-20 ..."></div>
<div class="**-mt-8** bg-light-blue-300 ...">-mt-8</div>
```

## Использование пользовательских значений

### Настройка вашей темы

По умолчанию для шкалы полей Tailwind используется [шкала интервалов по умолчанию](/docs/customizing-spacing). Вы можете настроить масштаб интервалов, отредактировав `theme.spacing` или `theme.extend.spacing` в вашем файле `tailwind.config.js`.

```diff-js tailwind.config.js
  module.exports = {
    theme: {
      extend: {
+       spacing: {
+         '5px': '5px',
+       }
      }
    }
  }
```

Кроме того, вы можете настроить только масштаб полей, отредактировав `theme.margin` или `theme.extend.margin` в вашем файле `tailwind.config.js`.

```diff-js tailwind.config.js
  module.exports = {
    theme: {
      extend: {
+       margin: {
+         '5px': '5px',
+       }
      }
    }
  }
```

Дополнительные сведения о настройке темы по умолчанию смотрите в документации [настройка темы](/docs/theme#customizing-the-default-theme).
