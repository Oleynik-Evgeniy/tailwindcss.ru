## Основы использования

### Adding a text indent

Use the `indent-{width}` utilities to set the amount of empty space (indentation) that's shown before text in a block.

<Example p="none">
  <div class="bg-white p-8 shadow-xl text-gray-700 max-w-md mx-auto overflow-auto leading-6">
    <p class="indent-8">
      Итак, я пошел в воду. I won't lie to you boys, I was
      terrified. But I pressed on, and as I made my way past the breakers
      a strange calm came over me. I don't know if it was divine intervention
      or the kinship of all living things but I tell you Jerry at that moment,
      I <em>was</em> a marine biologist.
    </p>
  </div>
</Example>

```html
<p class="**indent-8**">
  Итак, я пошел в воду. I won't lie to you boys, I was
  terrified. But I pressed on, and as I made my way past the breakers
  a strange calm came over me. I don't know if it was divine intervention
  or the kinship of all living things but I tell you Jerry at that moment,
  I <em>was</em> a marine biologist.
</p>
```

### Использование отрицательных значений

To use a negative text indent value, prefix the class name with a dash to convert it to a negative value.

```html
<div class="**-indent-8**">
  Итак, я пошел в воду. I won't lie to...
</div>
```

## Использование пользовательских значений

### Настройка вашей темы

The default text indent scale is based on the [default spacing scale](/docs/customizing-spacing#default-spacing-scale). Вы можете настроить масштаб интервалов, отредактировав `theme.spacing` или `theme.extend.spacing` в вашем файле `tailwind.config.js`.

```diff-js tailwind.config.js
  module.exports = {
    theme: {
      extend: {
+       spacing: {
+         '128': '32rem',
+       }
      }
    }
  }
```

Alternatively, you can customize just the text indent scale by editing `theme.textIndent` или `theme.extend.textIndent` в вашем файле `tailwind.config.js`.

```diff-js tailwind.config.js
  module.exports = {
    theme: {
      extend: {
+       textIndent: {
+         '128': '32rem',
+       }
      }
    }
  }
```

Дополнительные сведения о настройке темы по умолчанию смотрите в документации [настройка темы](/docs/theme#customizing-the-default-theme).
