## Основы использования

### Установка стиля границы

Используйте `border-{style}` для управления стилем границы элемента.

<Example>
  <div class="flex flex-col sm:flex-row items-center justify-around gap-4 text-white text-sm text-center font-bold leading-6">
    <div class="flex flex-col items-center shrink-0">
      <p class="font-medium text-sm text-gray-500 font-mono text-center mb-3">border-solid</p>
      <button class="px-4 py-2 font-semibold text-sm bg-white text-gray-700 rounded-md shadow-sm ring-1 ring-gray-900/5 border-indigo-500 border-2 border-solid">Button A</button>
    </div>
    <div class="flex flex-col items-center shrink-0">
      <p class="font-medium text-sm text-gray-500 font-mono text-center mb-3">border-dashed</p>
      <button class="px-4 py-2 font-semibold text-sm bg-white text-gray-700 rounded-md shadow-sm ring-1 ring-gray-900/5 border-indigo-500 border-2 border-dashed">Button A</button>
    </div>
    <div class="flex flex-col items-center shrink-0">
      <p class="font-medium text-sm text-gray-500 font-mono text-center mb-3">border-dotted</p>
      <button class="px-4 py-2 font-semibold text-sm bg-white text-gray-700 rounded-md shadow-sm ring-1 ring-gray-900/5 border-indigo-500 border-2 border-dotted">Button A</button>
    </div>
    <div class="flex flex-col items-center shrink-0">
      <p class="font-medium text-sm text-gray-500 font-mono text-center mb-3">border-double</p>
      <button class="px-4 py-2 font-semibold text-sm bg-white text-gray-700 rounded-md shadow-sm ring-1 ring-gray-900/5 border-indigo-500 border-4 border-double">Button A</button>
    </div>
  </div>
</Example>

```html
<div class="**border-solid** border-2 border-indigo-600 ..."></div>
<div class="**border-dashed** border-2 border-indigo-600 ..."></div>
<div class="**border-dotted** border-2 border-indigo-600 ..."></div>
<div class="**border-double** border-4 border-indigo-600 ..."></div>
```

### Без стиля

Используйте `border-none`, чтобы удалить существующий стиль границы из элемента.

Это чаще всего используется для удаления стиля границы, который был применен к меньшей контрольной точке.

<Example>
  <div class="flex justify-center gap-4 text-white text-sm text-center font-bold leading-6">
    <button class="px-4 py-2 font-semibold text-sm bg-white text-gray-700 rounded-md shadow-sm ring-1 ring-gray-900/5 border-indigo-500 border-2 border-none">Сохранить изменения</button>
  </div>
</Example>

```html
<button class="**border-none** ...">Сохранить изменения</button>
```
