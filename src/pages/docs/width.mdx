## Авто

Используйте `w-auto`, чтобы браузер мог рассчитать и выбрать ширину элемента. Вы можете использовать его, чтобы отключить определенную ширину:

```html
<div class="w-24 **md:w-auto** ..."></div>
```

## Ширина экрана

Используйте `w-screen`, чтобы элемент занимал всю ширину области просмотра.

```html
<div class="h-12 **w-screen**"></div>
```

## Адаптивность

Чтобы контролировать ширину элемента в определенной контрольной точке, добавьте префикс `{screen}:` к любой существующей утилите ширины. Например, добавление класса `md:w-full` к элементу приведет к применению утилиты `w-full` при средних размерах экрана и выше.

```html
<div class="w-1/2 **md:w-full** ...">
  <!-- ... -->
</div>
```

Для получения дополнительной информации о функциях адаптивного дизайна Tailwind ознакомьтесь с документацией [Адаптивного дизайна](/docs/responsive-design).

## Кастомизация

### Шкала ширины

По умолчанию масштаб ширины Tailwind представляет собой комбинацию [шкалы интервалов по умолчанию](/docs/customizing-spacing#default-spacing-scale), а также некоторых дополнительных значений, специфичных для ширины.

Вы можете настроить шкалу интервалов для заполнения, полей, ширины и высоты одновременно в разделах `theme.spacing` или `theme.extend.spacing` Вашего файла `tailwind.config.js`:

```diff-js
  // tailwind.config.js
  module.exports = {
    theme: {
      extend: {
+       spacing: {
+         '72': '18rem',
+         '84': '21rem',
+         '96': '24rem',
+       }
      }
    }
  }
```

Чтобы настроить ширину отдельно, используйте раздел `theme.width` Вашего файла `tailwind.config.js`.

```diff-js
  // tailwind.config.js
  module.exports = {
    theme: {
      extend: {
+       width: {
+         '1/7': '14.2857143%',
+         '2/7': '28.5714286%',
+         '3/7': '42.8571429%',
+         '4/7': '57.1428571%',
+         '5/7': '71.4285714%',
+         '6/7': '85.7142857%',
+       }
      }
    }
  }
```

Дополнительные сведения о настройке темы по умолчанию смотрите в [документации по настройке темы](/docs/theme#customizing-the-default-theme).
