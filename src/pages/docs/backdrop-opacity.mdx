## Применение

Используйте утилиты `backdrop-opacity-{amount}` вместе с утилитой `backdrop-filter` для управления непрозрачностью других фильтров фона, применяемых к элементу.

```html
<div class="backdrop-filter backdrop-opacity-80 ...">
  <!-- ... -->
</div>
```

## Адаптивность

Чтобы управлять прозрачностью фона элемента в определенной контрольной точке, добавьте префикс `{screen}:` к любой существующей утилите прозрачности фона. Например, используйте `md:backdrop-opacity-lg`, чтобы применить утилиту `backdrop-opacity-lg` только при средних размерах экрана и выше.

```html
<div class="backdrop-filter backdrop-opacity-sm **md:backdrop-opacity-lg** ...">
  <!-- ... -->
</div>
```

Для получения дополнительной информации о функциях адаптивного дизайна Tailwind ознакомьтесь с документацией [Адаптивного дизайна](/docs/responsive-design).

## Кастомизация

Вы можете настроить, какие утилиты `backdrop-opacity` будут генерироваться, используя ключ `backdropOpacity` в разделе `theme` вашего файла `tailwind.config.js`.

```diff-js
  // tailwind.config.js
  module.exports = {
    theme: {
+     extend: {
+       backdropOpacity: {
+         15: '.15',
+       }
+     }
    }
  }
```

Дополнительные сведения о настройке темы по умолчанию смотрите в [документации по настройке темы](/docs/theme#customizing-the-default-theme).
