## Применение

Управляйте цветом границы элемента с помощью утилит `border-{color}`.

```html rose
<template preview>
  <div class="max-w-xs w-full mx-auto">
    <input class="border border-red-500 bg-white text-gray-900 appearance-none block w-full rounded-md py-3 px-4 focus:outline-none" placeholder="Your email">
  </div>
</template>

<input class="border-2 **border-red-500** ...">
```

## Адаптивность

Чтобы управлять цветом границы элемента в определенной контрольной точке, добавьте префикс `{screen}:` к любой существующей утилите цвета границы. Например, используйте `md:border-green-500`, чтобы применить утилиту `border-green-500` только при средних размерах экрана и выше.

```html
<button class="border-blue-500 **md:border-green-500** ...">
  Button
</button>
```

Для получения дополнительной информации о функциях адаптивного дизайна Tailwind ознакомьтесь с документацией [Адаптивного дизайна](/docs/responsive-design).

## Наведение

Чтобы управлять цветом границы элемента при наведении, добавьте префикс `hover:` к любой существующей утилите цвета границы. Например, используйте `hover:border-blue-500`, чтобы применить утилиту `border-blue-500` при наведении курсора.

```html purple
<template preview>
  <div class="flex items-center justify-center">
    <button class="border-2 border-purple-500 hover:border-gray-500 bg-transparent text-purple-700 hover:text-gray-700 py-3 px-4 font-semibold rounded-md">
      Кнопка
    </button>
  </div>
</template>

<button class="border-2 border-purple-500 **hover:border-gray-500** ...">
  Кнопка
</button>
```

Утилиты Hover также можно комбинировать с адаптивными утилитами, добавив префикс `{screen}:` *перед* префиксом `hover:`.

```html
<button class="... md:border-blue-500 md:hover:border-blue-700 ...">Кнопка</button>
```

## Фокус

Чтобы управлять цветом границы элемента в фокусе, добавьте префикс `focus:` к любой существующей утилите цвета границы. Например, используйте `focus:border-blue-500`, чтобы применить утилиту `border-blue-500` к фокусу.

```html blue
<template preview>
  <div class="max-w-xs w-full mx-auto">
    <input class="border border-gray-300 bg-white text-gray-900 appearance-none block w-full rounded-md py-3 px-4 focus:border-blue-500 focus:outline-none" placeholder="Your email">
  </div>
</template>

<input class="border border-red-500 focus:border-blue-500 ...">
```

Утилиты Focus также можно комбинировать с адаптивными утилитами, добавляя префикс `{screen}:` *перед* префиксом `focus:`.

```html
<input class="... md:border-gray-200 md:focus:border-white ...">
```

## Кастомизация

### Цвета границы

По умолчанию Tailwind делает всю [цветовую палитру по умолчанию](/docs/customizing-colors#default-color-palette) доступной в качестве цветов границ.

Вы можете [настроить свою цветовую палитру](/docs/colors#customizing), отредактировав раздел `theme.colors` вашего файла `tailwind.config.js`, или настроить только цвета границ, используя раздел `theme.borderColor`.

```diff-js
  // tailwind.config.js
  module.exports = {
    theme: {
      borderColor: theme => ({
-       ...theme('colors'),
        DEFAULT: theme('colors.gray.300', 'currentColor'),
+       'primary': '#3490dc',
+       'secondary': '#ffed4a',
+       'danger': '#e3342f',
      })
    }
  }
```
