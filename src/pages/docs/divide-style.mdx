## Применение

Управляйте стилем границы между элементами с помощью утилит `divide-{style}`.

```html amber
<template preview>
  <div class="divide-y-4 divide-amber-600 divide-dashed">
    <div class="text-center font-extrabold text-2xl text-amber-600 py-3">1</div>
    <div class="text-center font-extrabold text-2xl text-amber-600 py-3">2</div>
    <div class="text-center font-extrabold text-2xl text-amber-600 py-3">3</div>
  </div>
</template>

<div class="divide-y-4 divide-yellow-600 **divide-dashed**">
  <div>1</div>
  <div>2</div>
  <div>3</div>
</div>
```

## Адаптивность

Чтобы управлять стилем границы между элементами в определенной контрольной точке, добавьте префикс `{screen}:` к любой существующей утилите стиля разделения. Например, используйте `md:divide-dashed`, чтобы применить утилиту `divide-dashed` только для средних размеров экрана и выше.

```html
<div class="divide-y-2 divide-dashed **md:divide-solid**">
  <!-- ... -->
</div>
```

Для получения дополнительной информации о функциях адаптивного дизайна Tailwind ознакомьтесь с документацией [Адаптивного дизайна](/docs/responsive-design).
