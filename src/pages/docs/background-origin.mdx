## Применение

## Адаптивность

Чтобы управлять свойством background-origin в определенной контрольной точке, добавьте префикс `{screen}:` к любой существующей утилите background-origin. Например, используйте `md:bg-origin-padding`, чтобы применить утилиту `bg-origin-padding` только при средних размерах экрана и выше.

```html
<div class="bg-origin-border **md:bg-origin-padding** ...">
  <!-- ... -->
</div>
```

Для получения дополнительной информации о функциях адаптивного дизайна Tailwind ознакомьтесь с документацией [Адаптивного дизайна](/docs/responsive-design).
