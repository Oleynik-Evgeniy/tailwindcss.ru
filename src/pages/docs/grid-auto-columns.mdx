## Применение

Используйте утилиты `auto-cols-{size}` для управления размером неявно создаваемых столбцов сетки.

```html
<div class="grid grid-flow-col **auto-cols-max**">
  <div>1</div>
  <div>2</div>
  <div>3</div>
</div>
```

## Адаптивность

Чтобы управлять свойством grid-auto-columns в определенной контрольной точке, добавьте префикс `{screen}:` к любой существующей утилите grid-auto-columns. Например, используйте `md:auto-cols-min`, чтобы применить утилиту `auto-cols-min` только при средних размерах экрана и выше.

```html
<div class="grid grid-flow-col auto-cols-max **md:auto-cols-min**">
  <div>1</div>
  <div>2</div>
  <div>3</div>
</div>
```

Для получения дополнительной информации о функциях адаптивного дизайна Tailwind ознакомьтесь с документацией [Адаптивного дизайна](/docs/responsive-design).

## Кастомизация

По умолчанию Tailwind включает в себя четыре универсальные утилиты с автоматическими столбцами сетки. Вы можете настроить их в разделе `theme.gridAutoColumns` Вашего файла `tailwind.config.js`.

```js
  // tailwind.config.js
  module.exports = {
    theme: {
      extend: {
        gridAutoColumns: {
          '2fr': 'minmax(0, 2fr)',
        }
      }
    }
  }
```

Дополнительные сведения о настройке темы по умолчанию смотрите в [документации по настройке темы](/docs/theme#customizing-the-default-theme).
